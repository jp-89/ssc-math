১.২১। $ n = 2x-1$ যেখানে x ∊ N, দেখাও যে, $n^2$ কে 8 (আট) দ্বারা ভাগ করলে প্রতিক্ষেত্রে 1 ভাগশেষ থাকবে।

সমাধান:
দেওয়া আছে,
n = 2x-1 [ যেখানে x ∊ N]
∴ n = 2x-1

উভয় পক্ষকে বর্গ করিয়া পাই,
$n^2 = (2x-1)^2 $
বা, $n^2 = 4x^2 -4x +1 $
বা, $n^2 = 4x(x -1)+1 $
বা, $ n^2 = 4x(x -1)+1 $

এখানে, লক্ষ্য করলে দেখা যায় যে $ x(x -1) $ পরপর দুটি ক্রমিক সংখ্যার গুনফল।
আমরা জানি, পরপর দুটি ক্রমিক সংখ্যার গুনফল 2 দ্বারা বিভাজ্য।

অতএব, $4x(x -1)$, সংখ্যাটি 8 (আট) দ্বারা বিভাজ্য।
সুতরাং,
প্রতীয়মান হয় যে, $ n^2 = 4x(x -1)+1 $ কে 8 (আট) দ্বারা ভাগ করলে প্রতিক্ষেত্রে 1 ভাগশেষ থাকবে।
