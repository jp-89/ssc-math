ক. ∠CAD এর পরিমান নির্ণয় কর।
সমাধান:
আমার জানি,
$∠ACB + ∠ACD = 180^0$
বা,$∠ACD = 180^0 -60^0$
বা,$∠ACD = 120^0$
আবার, ∆ACD এর ক্ষেত্রে,
$∠CAD + ∠ACD + ∠ADC = 180^0$ [ত্রিভুজের দুই কোনের সমষ্টি $180^0$ ]
বা, $∠CAD = 180^0 - ∠ACD ∠ADC $
বা, $∠CAD = 180^0 - 120^0 - 30^0 $
বা, $∠CAD = 30^0 $
বা,$∠CAD = 30^0$

খ. AB ও BC এর দৈর্ঘ্য নির্ণয় কর।
সমাধান:
আমার জানি,
${\text"লম্ব"}/{\text"অতিভূজ" } = sinθ$
বা, ${AB}/{AC} = sin60$
বা, ${AB}/{20} = {√3}/2$
বা, ${AB} = 20×{√3}/2$
বা, $AB = 10×√3$
বা,$AB= 10×√3$ মিটার


আমরা জানি,
${\text"ভূমি" }/{\text"অতিভুজ "} = cos θ$
বা, ${BC}/{AC} = cos60$
বা, ${BC}/{20} = cos60$
বা, ${BC} = 20×{1}/2$
বা, ${BC} = 10$
বা,$BC= 10$ মিটার

বিকল্প পদ্ধতি
আমরা জানি,সমকোণী ত্রিভুজের ক্ষেত্রে,
$AC^2 = AB^2 +BC^2$
বা,$AC^2 = AB^2 +BC^2$
বা,$BC^2 = AC^2 -AB^2$
বা,$BC^2 = 20^2 -(10√3)^2$
বা,$BC^2 =400 -300$
বা,$BC^2 =100$
বা,$BC =10$
