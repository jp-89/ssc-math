খুঁটির দৈর্ঘ্য BD = 64 মিটার।
মনে করি, A বিন্দুতে খুঁটিটি ভেঙে যায়।
ভাঙা অংশের দৈর্ঘ্য AD = AC = ?
ধরি, AC =x
ভূমির সাথে উন্নতি কোণ = $60^০$
AB = BD- AD
=$ BD- AC $
= $64- AC$
= $64-x$

আমরা জানি,
${\text"লম্ব "}/{\text"অতিভুজ "} = sinθ $
বা, $ {AB }/{AC } = sin60 $

বা, $ {64-x }/{x } = sin60 $

বা, $ {64-x}/{x } = {√3}/2 $

বা, $128-2x = √3x $

বা, $2x+√3x = 128 $

বা, $2x+1.732x = 128 $

বা, $3.732x = 128 $

বা, $x = {128}/{3.731} $

বা, $x = 34.298 $
