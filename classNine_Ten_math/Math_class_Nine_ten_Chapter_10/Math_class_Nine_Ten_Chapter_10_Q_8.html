গাছের চূড়ার উন্নতি = $60^০$
পাদদেশ থেকে স্থানের দুরুত্ব BC = 20 মিটার। গাছটির উচ্চতা AB = ?

আমরা জানি,
${\text"লম্ব "}/{\text"ভূমি"} = tanθ $
বা, $ {AB }/{BC } = tan60 $

বা, $ {AB }/{BC } = tan60 $

বা, $ {AB}/{20 } = tan60 $

বা, $AB = 20×tan60 $

বা, $AB = 20×√3 $

বা, $AB = 20×1.732 $

= 34.64 মিটার।
