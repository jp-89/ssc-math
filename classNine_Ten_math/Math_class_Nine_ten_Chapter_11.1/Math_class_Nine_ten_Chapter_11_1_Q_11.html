১১.১১। $x = {4ab}/{a+b} $ হলে দেখাও যে ,
$ {x+2a} /{x-2a} +{x+2b}/{x-2b} =2, a ≠ b$

সমাধান:
দেওয়া আছে,
$x = {4ab}/{a+b} $

বা, $x = {2a ×2b}/{a+b} $ ... (1)

উপরের সমীকরণকে নিম্নলিখিত আকারে প্রকাশ করা যায়
বা, $ x/{2a} = {2b}/{a+b} $ ... (3)

এবং
বা, $ x/{2b} = {2a}/{a+b} $ ... (4)

এখন (3) সমীকরণকে যোজন বিয়োজন করিয়া পাই
বা, $ {x+2a}/{x-2a} = {2b+a+b}/{2b -(a+b)} $

বা, $ {x+2a}/{x-2a} = {3b+a}/{b - a} $ ... (5)

এখন (4) সমীকরণকে যোজন বিয়োজন করিয়া পাই
বা, $ {x+2b}/{x-2b} = {2a+a+b}/{2a -(a+b)} $

বা, $ {x+2b}/{x-2b} = {3a+b}/{a-b} $ ... (6)

এখন (5) ও (6) সমীকরণ যোগ করিয়া পাই
বা, $ {x+2a}/{x-2a} + {x+2b}/{x-2b} = {3b+a}/{b - a} + {3a+b}/{a-b} $

বা, $ {x+2a}/{x-2a} + {x+2b}/{x-2b} = - {3b+a}/{a-b} + {3a+b}/{a-b} $

বা, $ {x+2a}/{x-2a} + {x+2b}/{x-2b} = {3a+b}/{a-b} - {3b+a}/{a-b} $

বা, $ {x+2a}/{x-2a} + {x+2b}/{x-2b} = {3a+b-3b-a}/{a-b} $

বা, $ {x+2a}/{x-2a} + {x+2b}/{x-2b} = {2a-2b}/{a-b} $

বা, $ {x+2a}/{x-2a} + {x+2b}/{x-2b} = {2(a-b)}/{a-b} $

বা, $ {x+2a}/{x-2a} + {x+2b}/{x-2b} = 2 $

নির্ণেয়: $ {x+2a}/{x-2a} + {x+2b}/{x-2b} = 2 $ (প্রমাণিত)

বিকল্প প্রমান
আমরা যদি $ {x+2a} /{x-2a} +{x+2b}/{x-2b} = 2 $ থেকে প্রমান করতে পারি,
$x = {4ab}/{a+b} $ তাহলে যথেষ্ট হবে।
দেওয়া আছে,
$ {x+2a} /{x-2a} +{x+2b}/{x-2b} = 2 $

 $ {(x+2a)(x-2b) + (x+2b)(x-2a) } /{(x-2a)(x-2b)} = 2 $

বা, $ {x^2-2bx +2ax -4ab + x^2-2ax+2bx-4ab } /{x^2-2bx-2ax+4ab} = 2 $

বা, $ {2x^2 -8ab } /{x^2-2bx-2ax+4ab} = 2 $

বা, $ {2(x^2 -4ab ) } /{x^2-2bx-2ax+4ab} = 2 $

বা, $ {(x^2 -4ab ) } /{x^2-2bx-2ax+4ab} = 1 $

বা, $ x^2 -4ab = x^2-2bx-2ax+4ab $

বা, $ 2bx+2ax = 4ab+4ab $

বা, $ 2bx+2ax = 8ab $

বা, $ bx+ax = 4ab $

বা, $ x(a+b) = 4ab $

বা, $ x = {4ab}/{a+b} $ (প্রমাণিত)
