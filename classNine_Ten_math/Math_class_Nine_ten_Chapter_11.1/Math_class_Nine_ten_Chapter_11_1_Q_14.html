১১.১৪। ${a^2 + b^2}/{b^2+c^2} = {(a+b)^2}/{(b+c)^2}$ হলে ,দেখাও যে ,
a, b, c ক্রমিক সমানুপাতি।

সমাধান:
দেওয়া আছে,
${a^2 + b^2}/{b^2+c^2} = {(a+b)^2}/{(b+c)^2}$

একান্তকরন করিয়া পাই
= ${a^2 + b^2}/{(a+b)^2} = {b^2+c^2}/{(b+c)^2}$

বা, $ {(a+b)^2}/{a^2 + b^2} = {(b+c)^2}/{b^2+c^2}$

বা, $ {a^+2ab+b^2}/{a^2 + b^2} = {b^2+2bc+c^2}/{b^2+c^2}$

বিয়োজন করিয়া পাই
বা, $ {(a^+2ab+b^2)-(a^2 + b^2 )}/{a^2 + b^2} = {(b^2+2bc+c^2)-(b^2+c^2)}/{b^2+c^2}$

বা, $ {a^+2ab+b^2-a^2 - b^2 }/{a^2 + b^2} = {b^2+2bc+c^2-b^2-c^2)}/{b^2+c^2}$

বা, $ {2ab }/{a^2 + b^2} = {2bc}/{b^2+c^2}$

বা, $ {a }/{a^2 + b^2} = {c}/{b^2+c^2}$

বা, $ ab^2 + c^2a = ca^2 + b^2c$

বা, $ c^2a - ca^2 = b^2c -ab^2$

বা, $ ca(c - a) = b^2(c -a)$

বা, $ ca = b^2 $

বা, $ a/b = b/c $

বা, $ a:b = b:c $

অতএব, a, b, c ক্রমিক সমানুপাতি।

নির্ণেয়: a, b, c ক্রমিক সমানুপাতি (প্রমাণিত )
