১১.১৭। $ {a+b-c}/{a+b} = {b+c-a}/{b+c}={c+a-b}/{c+a} $ এবংa+b+c ≠ 0 হলে, প্রমান কর যে ,
a = b= c

সমাধান:
দেওয়া আছে,
$ {a+b-c}/{a+b} = {b+c-a}/{b+c}={c+a-b}/{c+a} $

ধরি,
$ {a+b-c}/{a+b} = {b+c-a}/{b+c}={c+a-b}/{c+a} = k $

সুতরাং, আমরা পাই,
a+b-c = k(a+b) ....(1)
b+c-a = k(b+c) ....(2)
c+a-b = k(c+a) ....(3)

1,2 ও 3 সমীকরণ যোগ করিয়া পাই,
(a+b-c) + (b+c-a)+ (c+a-b) = k(a+b) + k(b+c) + k(c+a)

বা, (a+b+c)= k(a+b+b+c+c+a)

বা, (a+b+c)= k(2a+2b+2c)

বা, (a+b+c)= 2k(a+b+c)

বা, 1=2k

বা, k = $1/2$

(1) সমীকরণ থেকে পাই
a+b-c = k(a+b)
k এর মান বসিয়ে পাই
$ a+b-c = 1/2(a+b)$

বা, $ 2a+2b-2c = a+b $
বা, $ a+b = 2c $ ....(4)

(2) সমীকরণ থেকে পাই
b+c-a = k(b+c) )
এর k মান বসিয়ে পাই
$ b+c-a = 1/2(b+c)$

বা, $ 2b+2c-2a = b+c $
বা, $ b+c = 2a $

বা, $ 2b+2c = 4a $

(4) সমীকরণ থেকে 2c= a+b মান বসিয়ে পাই
বা, $ 2b+a+b = 4a $

বা, $ 3b = 3a $

বা, $ b = a $

(4) সমীকরণ এ a =b মান বসিয়ে পাই
বা, $ b+b = 2c $

বা, $ 2b = 2c $

বা, $ b = c $

নির্ণেয়: $ a = b = c $
