১১.২। একটি বৃত্তক্ষেত্রের ক্ষেত্রফল একটি বর্গক্ষেত্রের ক্ষেত্রফলের সমান হলে, তাদের পরিসীমার অনুপাত নির্ণয় কর
 সমাধান:
মনে করি, বর্গক্ষেত্রের বাহু = a এবং
বৃত্তের ব্যাসার্ধ = r
অতএব,
বর্গক্ষেত্রের ক্ষেত্রফল = $a^2$
বৃত্তের ক্ষেত্রফল = $ πr^2$

শর্তানুসারে,
$a^2 = πr^2$
বা, $ a = √{πr^2} $

আবার বর্গক্ষেত্রের পরিসীমা = 4a এবং
বৃত্তের পরিসীমা = 2πr

সুতরাং,
পরিসীমার অনুপাত= $\text"বৃত্তের পরিসীমা "/\text"বর্গক্ষেত্রের পরিসীমা "$
= ${2πr}/ {4a} $

$ a = √{πr^2} $ মান বসিয়ে পাই,
পরিসীমার অনুপাত = ${2 πr}/ {4(√{πr^2})} $

= ${2πr}/ {4(√π r^)} $

= ${ π}/ {2√π } $

= ${ √π}/ {2 } $

পরিসীমার অনুপাত: $√π : 2 $
