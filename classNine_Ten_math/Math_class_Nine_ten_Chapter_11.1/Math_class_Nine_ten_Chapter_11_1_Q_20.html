১১.২০। যদি (a+b+c)p = (b+c-a)q = (c+a-b)r = (a+b-c)s হয়, তবে প্রমান কর যে
$ 1/q +1/r + 1/s = 1/p$

সমাধান:
দেওয়া আছে,
(a+b+c)p = (b+c-a)q = (c+a-b)r = (a+b-c)s

ধরি,
(a+b+c)p = (b+c-a)q = (c+a-b)r = (a+b-c)s = k

সুতরাং, আমরা পাই,
$ (a+b+c)p = k $
বা, $ 1/p = {(a+b+c)}/k $

আবার,
$ (b+c-a)q = k $
বা, $ 1/q = {(b+c-a)}/k $

আবার,
$ (c+a-b)r = k $
বা, $ 1/r = {(c+a-b)} /k$

আবার,
$ (a+b-c)s = k $
বা, $ 1/s = {(a+b-c)}/k $

দেওয়া আছে,
বামপক্ষ = = $ 1/q +1/r + 1/s$

মান বসিয়ে পাই,
= ${(b+c-a)}/k + {(c+a-b)} /k + {(a+b-c)}/k $

= ${(b+c-a) + (c+a-b)+ (a+b-c) }/k $

= ${b+c-a + c+a-b+ a+b-c }/k$

= ${ a+b + c}/k$

উপরে প্রাপ্ত মান থেকে পাই
= $1/p$

নির্ণেয়: বামপক্ষ = ডানপক্ষ (প্রমানিত )
