১১.২১। যদি $ lx = my = nz $ হয়, তবে দেখাও যে
$ x^2/{yz} + y^2/{zx} + z^2/{xy} = {mn}/l^2 + {nl}/m^2 + {lm}/n^2$

সমাধান:
দেওয়া আছে,
$ lx = my = nz $

ধরি,
$ lx = my = nz = k $

সুতরাং, আমরা পাই,
$ lx = k $
বা, $ x = k/l $

আবার,
$ my = k $
বা, $ y = k/m $

আবার,
$ nz = k $
বা, $ z = k/n $

দেওয়া আছে,
বামপক্ষ = = $ x^2/{yz} + y^2/{zx} + z^2/{xy}$

মান বসিয়ে পাই,
= ${(k/l)^2}/{( k/m)( k/n)} +{(k/m)^2}/{( k/n)( k/l )} + {( k/n)^2}/{( k/l )( k/m)}$

= ${k^2/l^2}/{k^2/{mn}} +{k^2/m^2}/{ k^2/{nl}} + {k^2/n^2}/{ k^2/{lm }}$

= ${mn}/l^2 +{nl}/m^2 + {lm }/n^2$

নির্ণেয়: বামপক্ষ = ডানপক্ষ (প্রমানিত )
