১১. ৭। যদি a:b = b:c হয়, তবে প্রমান কর যে,
(i) $ a/c = {a^2 + b^2}/ {b ^2 + c^2 } $
সমাধান:
দেওয়া আছে,
a:b = b:c
বা, $ a /b = b /c $
দেওয়া আছে,
ধরি,
$ a /b = b /c $ = k
অতএব, আমরা পাই,
$ b = ck $ এবং
$ a = bk $
বা, $ a = (ck)k $
বা, $ a = ck^2 $

এখন, বামপক্ষ = $ a/c $
মান বসিয়ে পাই,
= ${ck^2}/{c} $
= $k^2$

আবার, ডানপক্ষ = $ {a^2 + b^2}/ {b ^2 + c^2 } $
মান বসিয়ে পাই,
=$ {(ck^2)^2 + (ck)^2}/ {(ck) ^2 + c^2 } $

=$ {k^2(c^2k^2 + c^2)}/ {(c^2k ^2 + c^2 )} $

=$k^2$

নির্ণেয় : বামপক্ষ = ডানপক্ষ (প্রমাণিত )

১১. ৭। যদি a:b = b:c হয়, তবে প্রমান কর যে,
(ii) $a^2b^2c^2 ( 1/a^3 + 1/b^3 +1/c^3) = a^3+b^3+c^3 $

সমাধান:
দেওয়া আছে,
a:b = b:c
বা, $ a /b = b /c $
বা, $b^2 = ac $
দেওয়া আছে,
বামপক্ষ =$a^2b^2c^2 ( 1/a^3 + 1/b^3 +1/c^3)$

= ${a^2b^2c^2}/a^3 + {a^2b^2c^2}/b^3+ {a^2b^2c^2}/c^3$

= ${b^2c^2}/a + {a^2c^2}/b+ {a^2b^2}/c$

মান বসিয়ে পাই,
= ${(ac)c^2}/a + {(b^2)^2}/b+ {a^2(ac)}/c$

= ${ac^3}/a + {b^4}/b+ {a^3c}/c$

= $c^3 + b^3+ a^3 $

= $a^3 + b^3+ c^3 $

নির্ণেয় : বামপক্ষ = ডানপক্ষ (প্রমাণিত )

১১. ৭। যদি a:b = b:c হয়, তবে প্রমান কর যে,
(iii)$ {abc(a+b+c)^3}/{(ab+bc+ca)^3} =1 $
সমাধান:
দেওয়া আছে,
a:b = b:c
বা, $ a /b = b /c $
বা, $b^2 = ac $
দেওয়া আছে,
বামপক্ষ =$ {abc(a+b+c)^3}/{(ab+bc+ca)^3}$

$b^2 = ac $ মান বসিয়ে পাই,
= $ {abc(a+b+c)^3}/{(ab+bc+b^2)^3} $

= $ {abc(a+b+c)^3}/{\{b(b+c+b)\}^3}$

= $ {abc(a+b+c)^3}/{b^3{(b+c+b)^3}$

= $ {abc}/b^3 $

= $ {acb}/b^3 $

= $ {b^2b}/b^3 $

= $ {b^3}/b^3 $

= 1
নির্ণেয় : বামপক্ষ = ডানপক্ষ (প্রমাণিত )

= $ ck^2-2(ck)+c $
= $ ck^2-2ck+c $
= $ c(k^2-2k+1) $
= $ c(k-1)^2 $

= $ {\{ck(k-1)\}^2}/{ck^2} $
=$ {c^2k^2(k-1)^2}/{ck^2} $
= $ c(k-1)^2 $
=

তৃতীয় রাশি = ${(b-c)^2}/{c } $
মান বসিয়ে পাই,
