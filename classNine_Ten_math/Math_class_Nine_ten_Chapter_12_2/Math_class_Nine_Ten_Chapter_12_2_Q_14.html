১২.২.১৪। আড়গুণন পদ্ধতিতে সমাধান কর:
$y(3+x) = x(6+y) $
$ 3(3+x) =5(y-1) $

সমাধান:
দেওয়া আছে,
$y(3+x) = x(6+y) $
$ 3(3+x) =5(y-1) $
y(3+x) = x(6+y) সমীকরণকে প্রয়োজনমত সাজিয়ে লিখলে পাই
$ 3y+xy = 6x+xy $
$6x-3y + 0 = 0 $
এবং, 3(3+x) =5(y-1) সমীকরণকে প্রয়োজনমত সাজিয়ে লিখলে পাই
$ 9+3x = 5y-5 $
$3x-5y +14 =0 $

অতএব, সমীকরণ দুইটিকে প্রয়োজনমত সাজিয়ে লিখলে পাই
$ 6x-3y + 0 = 0 $ ...(1)
$ 3x-5y +14 = 0 $ ...(2)
আড়গুণন পদ্ধতি প্রয়োগ করিয়া পাই,
$ x/{(3)×(14) -(-5)×(0)} = y/{(0)×3-(14)×6}= 1/{(6)×(-5)-(3)×(-3)} $

বা, $ x/{-42} = y/{-84} = 1/{-30+9} $

বা, $ x/{-42} = y/{-84} = 1/{-21} $

অতএব,
$ x/{-42} ={1}/{-21} $

বা, $ x ={-42} /{-21}$

বা, $ x = 2$ এবং

$ y/{-84} = 1/{-21} $

বা, $ y = {-84}/{-21} $

বা, $ y = 4 $
