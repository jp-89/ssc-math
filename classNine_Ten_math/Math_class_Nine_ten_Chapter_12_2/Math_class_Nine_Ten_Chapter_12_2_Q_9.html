১২.২.৯। আড়গুণন পদ্ধতিতে সমাধান কর:
$ x + 2y = 7 $
$ 2x - 3y = 0 $

সমাধান:
দেওয়া আছে,
$ x + 2y = 7 $ ...(1)
$ 2x - 3y = 0 $ ...(2)
সমীকরণ দুইটিকে প্রয়োজনমত সাজিয়ে লিখলে পাই
$ x + 2y -7 = 0 $ ...(3)
$ 2x - 3y + 0 = 0 $ ...(4)
আড়গুণন পদ্ধতি প্রয়োগ করিয়া পাই,
$ x/{2×0 -(-3)×(-7)} = y/{(-7)×2-0×1}= 1/{1×(-3)-2×2} $

বা, $ x/{0-21} = y/{-14-0} = 1/{-3-4} $

বা, $ x/{-21} = y/{-14} = 1/{-7} $

অতএব,
$ x/{-21} ={1}/{-7} $

বা, $ x = {-21} /{-7}$

বা, $ x = {3}$ এবং
$ y/{-14} = 1/{-7} $

বা, $ y = {-14}/{-7} $

বা, $ y = 2 $

নির্ণেয়:
 $ x = 3 $ এবং
$ y = 2 $
