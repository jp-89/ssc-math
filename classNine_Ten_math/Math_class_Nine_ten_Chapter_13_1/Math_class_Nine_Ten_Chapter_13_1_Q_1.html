১৩.১.১। 2 -5 - 7 -19 ... ধারাটির অন্তর ও 12 তম পদ নির্ণয় কর।
সমাধান:
ধারাটি এভাবে লিখতে পারি 2 + ( -5 ) + ( - 7 ) +( -19 ) ...

2 + ( -5 ) + ( - 7 ) + ( -19 ) ... একটি সমান্তর ধারা
ধারাটির প্রথম পদ a = 2
সাধারন অন্তর d = (-5) -2 = -7
ধরি, ধারাটির 12 তম পদ = ?

আমরা জানি, সমান্তর ধারার n তম পদ = a +(n -1)d
এখানে n = 12
অতএব, 12 তম পদ = a +(n -1)d
= 2 +(12 -1)(-7)
= 2 + 11 (-7)
= 2-77
= -75;

নির্ণেয়:
 সাধারন অন্তর = -7 এবং
12 তম পদ = -75
