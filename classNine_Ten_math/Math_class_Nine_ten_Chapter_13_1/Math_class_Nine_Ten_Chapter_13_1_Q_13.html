১৩.১.১৩। 2 + 4 + 6 + 8 + .... .... ধারাটির প্রথম n পদের যোগফল 2550 হলে , n এর মান নির্ণয় কর।
সমাধান:
ধরি,সমান্তর ধারার সাধারণ অন্তর d = 4 -2 = 2
এবং প্রথম পদ a = 2

আমরা জানি সমান্তর ধারার n তম পদের সমষ্টি = $ n/2 \{2a + (n-1)d \} = 2550 $
a এবং d এর মান বসিয়ে পাই,
$ n/2 \{2a + (n-1)d \} = 2550 $
বা, $ n/2 \{2×2 + (n-1)×(2) \} = 2550 $
বা, $ n/2 (4 +2n-2) = 2550 $
বা, $ n/2 \(2n +2) = 2550 $
বা, $ n/2 × 2(n+1) = 2550 $
বা, $ n(n+1) = 2550 $
বা, $n^2 + n -2550 = 0 $
বা, $n^2 -50n +49n -2550 = 0 $
বা, $n(n -50) + 49(n-50) = 0 $
বা, $( n-50)(n+49) = 0 $
বা, $( n-50) = 0 $
n= 50

আবার, $n+49 = 0 $
বা, n= -49
n এর মান নিগেটিভ হতে পারে না।
