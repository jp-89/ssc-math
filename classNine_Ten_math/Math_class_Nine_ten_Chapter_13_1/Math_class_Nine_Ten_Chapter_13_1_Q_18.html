১৩.১.১৮। কোনো সমান্তর ধারার p তম, q তম ও r তম পদ যথাক্রমে a, b, c হলে, দেখাও যে , a(q -r ) + b( r-p) + c(p -q) = 0
সমাধান:
ধরি,সমান্তর ধারার সাধারণ অন্তর = d
এবং প্রথম পদ = m

আমরা জানি, সমান্তর ধারার n তম পদ = m +(n -1)d

অতএব, p তম পদ = $ m +(p -1)d = a $
বা, $ m + (p -1 )d = a $ ......(1)

q তম পদ = $ m +(q -1)d = b $ .........(2)

r তম পদ = $ m +(r -1)d = c $ .........(3)

(1) সমীকরণ থেকে (2) সমীকরণ বিয়োগ করিয়া পাই,
$(p -1 )d - (q -1)d = a-b $
বা, $(p -q )d = a-b $

বা, $d = {(a-b)}/ {(p -q )}$ ........ (4)

(2) সমীকরণ থেকে (3) সমীকরণ বিয়োগ করিয়া পাই,
$(q -1 )d - (r -1)d = b -c $
বা, $(q -r )d = b -c $

বা, $d = {(b - c)}/ {(q -r )}$ ........ (5)

(4) সমীকরণ ও (5) সমীকরণ থেকে পাই,
$ {(a-b)}/ {(p -q )} = {(b - c)}/ {(q -r )} $

বা, $ (a-b)(q -r ) = (p -q )(b - c)$

বা, $ a(q -r ) -b(q -r ) = b(p -q ) - c(p -q )$

বা, $ a(q -r ) + c(p -q ) = b(p -q ) + b(q -r )$ [পক্ষান্তর করিয়া ]

বা, $ a(q -r ) + c(p -q ) = b(p -q +q -r )$

বা, $ a(q -r ) + c(p -q ) = b(p -r )$

বা, $ a(q -r ) + c(p -q ) = -b(r -p )$

বা, $ a(q -r ) + c(p -q ) + b(r -p ) = 0 $ [পক্ষান্তর করিয়া ]

বা, $ a(q -r ) + b(r -p )+ c(p -q ) = 0 $


নির্ণেয়:
 $ a(q -r ) + b(r -p )+ c(p -q ) = 0 $( প্রমাণিত )
