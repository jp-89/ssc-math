১৩.১.১৯। দেখাও যে , 1 + 3+ 5 +7 + ........+ 125 = 169 + 171 + 173 + .... + 209
সমাধান:
1 + 3+ 5 +7 + ........+ 125 একটি সমান্তর ধারা
ধারাটির প্রথম পদ a = 1
ধারাটির শেষ পদ = 125
সাধারন অন্তর d = 3-1 = 2
ধরি, ধারাটির n তম পদ = 125
আমরা জানি সমান্তর ধারার n তম পদ = a +(n -1)d
অতএব , a +(n -1)d = 125
বা, 1+ (n -1)2 = 125;
বা, $(n -1) = {125 -1}/2 $
বা, $ n -1 = {124}/2 $
বা, n = 62 +1;
বা, n = 63

ধারাটির সমষ্টি $ S = n/2\{2a+(n-1)d\} $

=$ {63}/2\{2*1+(63-1)2\} $

=$ {63}/2\{2+62*2\} $

=$ {63}/2\{2(1+62)\} $

=$ 63*63 $

= $3969 $

আবার ,
169 + 171 + 173 + .... + 209 একটি সমান্তর ধারা
ধারাটির প্রথম পদ a = 169
ধারাটির শেষ পদ = 209
সাধারন অন্তর d = 171-169 = 2
ধরি, ধারাটির n তম পদ = 209;
আমরা জানি সমান্তর ধারার n তম পদ = a +(n -1)d
অতএব, 169 +(n -1)d = 209
বা, 169+ (n -1)2 = 209;
বা, $(n -1) = {209 -169}/2 $
বা, $ n -1 = {40}/2 $
বা, n = 20 +1;
বা, n = 21

ধারাটির সমষ্টি $ S = n/2\{2a+(n-1)d\} $

=$ {21}/2\{2*169+(21-1)2\} $

=$ {21}/2\{2*169+20*2\} $

=$ {21}/2\{2(169+20)\} $

=$ 21*189 $

= $3969 $
