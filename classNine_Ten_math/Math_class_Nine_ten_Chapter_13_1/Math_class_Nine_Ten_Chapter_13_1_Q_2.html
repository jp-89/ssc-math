১৩.১.২। 8 + 11 + 14+ 17 + ... + ধারাটির কোন পদ 392
সমাধান:
8 + 11 + 14+ 17 + ... + একটি সমান্তর ধারা
ধারাটির প্রথম পদ a = 8
সাধারন অন্তর d = 11-8 = 3
ধরি, ধারাটির n তম পদ =392

আমরা জানি সমান্তর ধারার n তম পদ = a +(n -1)d
অতএব , a +(n -1)d = 392
বা, 8 + (n -1)3 = 392;
বা, $(n -1)3 = {392 -8} $
বা, $ n -1 = {384}/3 $
বা, n = 128 +1;
বা, n = 129

নির্ণেয়:
 129 তম পদ = 392
