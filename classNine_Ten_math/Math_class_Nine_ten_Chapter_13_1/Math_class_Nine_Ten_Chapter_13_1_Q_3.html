১৩.১.৩। 4 + 7 + 10+ 13 + ... + ধারাটির কোন পদ 301
সমাধান:
4 + 7 + 10+ 13 + ... + একটি সমান্তর ধারা
ধারাটির প্রথম পদ a = 4
ধারাটির শেষ পদ = 125
সাধারন অন্তর d = 7-4 = 3
ধরি, ধারাটির n তম পদ = 301

আমরা জানি সমান্তর ধারার n তম পদ = a +(n -1)d
অতএব , a +(n -1)d = 301
বা, 4 + (n -1)3 = 301;
বা, $(n -1)3 = {301 -4} $
বা, $ n -1 = {297}/3 $
বা, n = 99 +1;
বা, n = 100

নির্ণেয়:
 100 তম পদ = 301
