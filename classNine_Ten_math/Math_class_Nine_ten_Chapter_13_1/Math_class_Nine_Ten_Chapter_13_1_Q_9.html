১৩.১.৯ । 29 + 25 + 21+ ... -23 =কত?
সমাধান:
29 + 25 + 21+ ... -23 একটি সমান্তর ধারা
ধারাটির প্রথম পদ a = 29
ধারাটির n তম পদ p = -23
সাধারন অন্তর d = 29-25 = -4
n = ?
ধরি, ধারাটির প্রথম n টি পদের সমষ্ট = ?

আমরা জানি, সমান্তর ধারার n তম পদ = a +(n -1)d
এখানে n তম পদ = -23
-23 = 29 +(n -1)(-4)
বা, $ (n -1)(-4) = -23-29 $
বা, $ n -1 = {-52}/{-4} $

বা, $ n -1 = 13 $
বা, $ n = 14 $

আমরা জানি, সমান্তর ধারার n সংখ্যক স্বাভাবিক সংখ্যার সমষ্টি,
$ S_n = n/2 \{{2a+(n-1)d}\} $
অতএব,
$ S_n = {14}/2 \{{2.29+(14-1)(-4)}\} $
=$7 \{{58-52}\} $
=$7 \{{6\} $
=$42 $
