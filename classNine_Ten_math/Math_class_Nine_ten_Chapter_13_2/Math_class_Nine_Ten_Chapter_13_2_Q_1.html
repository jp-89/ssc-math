ক $ b = {c+d}/2 $
খ $ a= {b+c }/2$
গ $ c= { b+d } /2 $
ঘ $ d= { a+c } /2$

সমাধান:
যেহেতু a,b,c ও d একটি সমান্তর ধারার চারটি ক্রমিক পদ

a= a
b = a+1
c= b+1 = a+1+1=a+2
d = c+1= a+2+1=a+3
এখন প্রদত্ত সমীকরণ গুলো যাচাই করে দেখি।

= ${ 2a+5}/2$

≠ (a+1)
