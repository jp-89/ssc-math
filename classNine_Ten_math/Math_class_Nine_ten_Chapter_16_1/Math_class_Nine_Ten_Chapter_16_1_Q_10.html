সমাধান:
ABC একটি সমবাহু ত্রিভুজ।
অতএব,
AB = BC = CA
মনে করি,
O, ত্রিভুজের অভ্যন্তরস্থ একটি বিন্দু।
OD, OE, OF, O বিন্দু থেকে যথাক্রমে BC, CA এবং AB এর উপর অঙ্কিত লম্ব।
মনে করি,
OD = 8 সে.মি.
OE = 7 সে.মি.
OF = 6 সে.মি.
O,A এবং O,B এবং O,C যোগ করি।
ফলে, ABC ত্রিভুজটি ∆BOC, ∆AOB, ∆AOC এ বিভক্ত হয়ে যায়।
∆ ABC এর ক্ষেত্রফল =
∆ BOC এর ক্ষেত্রফল
+ ∆ AOB এর ক্ষেত্রফল
+ ∆ AOC এর ক্ষেত্রফল
আমরা জানি,
ত্রিভুজের ক্ষেত্রফল =$ 1/2 ×{ ভূমি}×{ উচ্চতা } $
অতএব,
BOC এর ক্ষেত্রফল = $1/2×{BC}×{OD}$
 = $1/2×{BC}×{8}$
= $4×{BC}$
AOB এর ক্ষেত্রফল = $1/2×{AB}×{OF}$
= $1/2×{BC}×{6}$
= $3×{BC}$
AOC এর ক্ষেত্রফল = $1/2×{AC}×{OE}$
= $1/2×{BC}×{7}$
= $3.5×{BC}$

∆ ABC এর ক্ষেত্রফল = $4×{BC}$ + $3×{BC}$ +$3.5×{BC}$
= $10.5×{BC}$

যেহেতু ত্রিভুজটি সমবাহু, সেহেতু প্রত্যেকটি কোনই $60^০$
সেক্ষেত্রে,
∆ ABC এর ক্ষেত্রফল = $1/2×{AB} ×BC ×sin60 $
= $1/2×{BC} ×BC ×sin60 $
= $1/2×{BC} ×BC ×{√3}/2 $
= ${√3}/4×{(BC)^2} $

শর্তানুসারে,
${√3}/4×{(BC)^2} = 10.5×{BC} $
বা, ${√3}/4×{BC} = 10.5 $
বা, ${BC} = {10.5 ×4}/{√3} $
বা, ${BC} = 24.249 $ সে.মি

∆ ABC এর ক্ষেত্রফল = $10.5×{BC}$
= $10.5×{24.249}$
= 254.611 বর্গ সে.মি (প্রায় )
