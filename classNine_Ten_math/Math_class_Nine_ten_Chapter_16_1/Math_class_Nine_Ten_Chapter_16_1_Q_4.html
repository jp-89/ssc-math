দুইটি বাহুর দৈর্ঘ্য 25 সে.মি.,27 সে.মি
পরিসীমা 2s = AB +AC +BC = 84 সে.মি।
মনে করি,
AB = c = 25 সে.মি
AC = b = 27 সে.মি
অতএব,
BC = a= 84-25-27 = 32 সে.মি
আমরা জানি,
ত্রিভুজের পরিসীমা 2s = a+b+c
বা, 2s = 32+27+25
বা, s = $ {84}/2$
বা, s = $ {42}$

ত্রিভুজের ক্ষেত্রফল = $√{s(s-a)(s-b)(s-c)}$
= $√{42(42-32)(42-27)(42-25)}$
= $√{42×10×15×17}$
= $√{107100}$
= 327.26 বর্গ সে.মি (প্রায়)
