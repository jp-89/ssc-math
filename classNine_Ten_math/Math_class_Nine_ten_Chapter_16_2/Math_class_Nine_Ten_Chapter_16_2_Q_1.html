ধরি,
বাগানের বিস্তার = x মিটার
সুতরাং, দৈর্ঘ্য = 2x মিটার
বাগানের ক্ষেত্রফল = $ 2x ×x $ বর্গমিটার
= $2x^2$ বর্গমিটার

শর্তানুসারে,
$2x^2 = 512 $
বা, $x^2 = {512}/2 $
বা, $x^2 = 256 $
বা, $x = 16 $
অতএব, বাগানের বিস্তার = 16 মিটার
বাগানের দৈর্ঘ্য= $2×16 = 32$ মিটার

বাগানের পরিসীমা = $2× ( দৈর্ঘ্য + বিস্তার ) $
= $2× ( 32 +16) $
= $2× 48 $
= 96
অতএব, বাগানের পরিসীমা =96 মিটার।
