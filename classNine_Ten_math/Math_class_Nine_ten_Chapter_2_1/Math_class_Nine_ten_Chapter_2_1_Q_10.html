২.১.১০। যে সকল স্বাভাবিক সংখ্যা দ্বারা 346 এবং 556 কে ভাগ করলে প্রতিক্ষেত্ৰে 31 অবশিষ্ঠ থাকে, এদের সেট নির্ণয় কর।

সমাধান:
  • যে সকল স্বাভাবিক সংখ্যা দ্বারা 346 এবং 556 কে ভাগ করলে প্রতিক্ষেত্ৰে 31 অবশিষ্ঠ থাকে,সে স্বাভাবিক সংখ্যাগুলো অবশ্যই 31 অপেক্ষা বড় হবে।
  • (346-31) = 315 এবং (556-31) = 525 সংখ্যাদ্বয় অবশ্যই সে স্বাভাবিক সংখ্যাগুলো দ্বারা বিভাজ্য হবে।
  • সে স্বাভাবিক সংখ্যাগুলো অবশ্যই 315 এবং 525 এর সাধারণ গুণনীয়ক হবে।

ধরি,
A= 315
B = 525

A = 315 কে নিম্নোক্ত গুণনীয়কের আকারে প্রকাশ করতে পারি।
315 = $1×315 $
= $3×105 $
= $5×63 $
= $7×45 $
= $9×35 $
অতএব, 315 এর গুণনীয়ক C = {1,3,5,7,9,35 45,63,105,315 }
অনুরূপভাবে,
B = 525 কে নিম্নোক্ত গুণনীয়কের আকারে প্রকাশ করতে পারি।
45 = $1×525 $
= $3×175 $
= $5×105 $
= $7×75 $
= $15×35 $

অতএব, 525 এর গুণনীয়ক D = {1,3,5,7,15,35,75,105,175,525}
এখন,
315 এবং 525 এর সাধারণ গুণনীয়ক = C এবং D এর সাধারণ গুণনীয়ক
= C ∩ D

C ∩ D = উভয় সেটের সাধারণ উপাদান
{1,3,5,7,9,35 45,63,105,315 } ∩ {1,3,5,7,15,35,75,105,175,525}
= {1,3,5,7,35,105}

কিন্ত শর্তানুসারে সাধারণ গুণনীয়ক অবশ্যই 31 অপেক্ষা বড় হবে
সুতরাং সাধারণ গুণনীয়ক = {35, 105}
