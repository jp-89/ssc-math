১। নিচের সেট গুলোকে সেট গঠন পদ্ধতিতে প্রকাশ কর।
ক) {3, 5, 7, 9, 11}
সমাধান:
{3, 5, 7, 9, 11} সেটের উপাদানগুলো প্রত্যেকটি বিজোড় সংখ্যা।
এবং কোন সংখ্যা 3 থেকে ছোট নয় এবং11 থেকে বড় নয়।
3 এর পূর্ববর্তী বেজোড় সংখ্যাটি হল 1 এবং 11 এর পরবর্তী বেজোড় সংখ্যাটি হল 13.

∴ নির্ণেয় সেট={ $ x ∈ N:x \text" বেজোড় সংখ্যা এবং " 1< x < 13$ }( উত্তর )

১। নিচের সেট গুলোকে সেট গঠন পদ্ধতিতে প্রকাশ কর।
খ) {1, 2, 3, 4, 6, 9, 12, 18, 36}
সমাধান:
{1, 2, 3, 4, 6, 9, 12, 18, 36} সেটের উপাদানগুলো প্রত্যেকটি 36 এর গুণনীয়ক
∴ নির্ণেয় সেট={ $ x ∈ N:x \text", 36 এর গুণনীয়ক " $ }( উত্তর )

১। নিচের সেট গুলোকে সেট গঠন পদ্ধতিতে প্রকাশ কর।
গ) {4, 8, 12, 16, 20, 24, 28, 32, 40}
সমাধান:
{4, 8, 12, 16, 20, 24, 28, 32, 40} সেটের উপাদানগুলো প্রত্যেকটি 4 এর গুণিতক এবং 40 এর বড় নয়
∴ নির্ণেয় সেট={ $ x ∈ N:x \text", 4 এর গুণিতক এবং " x ≤ 40$ }( উত্তর )

১। নিচের সেট গুলোকে সেট গঠন পদ্ধতিতে প্রকাশ কর।
ঘ ) { ${±4, ±5, ±6}$}
সমাধান:
প্রশ্নটির সমাধানের জন্য x এর প্রদত্ত বিভিন্ন মানের জন্য $x^2 $এবং $x ^3 $ মান নিম্নরূপ বের করি।
  x    $x^2$     $x^3$ 
  -4    16    -64 
  4    16    64 
  -5    25    -125 
  5    25    125 
  -6    36    -216 
  6    36    216 

উপরের তালিকা থেকে আমরা পাই ,
সেটের উপাদান সমূহের 16 বর্গ অপেক্ষা বড় অথবা সমান
সেটের উপাদান সমূহের ঘন 216 অপেক্ষা ছোট অথবা সমান
∴ নির্ণেয় সেট= { $ x ∈ Z: x^2 ≥ 16 \text" এবং " x ^3 ≤ 216$ } ( উত্তর )
