২.১.৩। A = {2,3,4}, B= {1,2,a} এবং C= {2,a,b} হলে, সেটগুলো নির্ণয় কর।
(ক) B\C
সমাধান:
দেওয়া আছে,
A = {2,3,4}
B= {1,2,a}
C= {2,a,b}

B\C - এর অৰ্থ হল, যে সমস্ত উপাদান B সেটে আছে কিন্ত C সেটে নেই এরূপ উপাদান নিয়ে গঠিত সেট।
অর্থাৎ B সেট থেকে C সেটের উপাদান গুলো বাদ দিলে অবশিষ্ট উপাদান নিয়ে গঠিত সেট।
অর্থাৎ B\C = B-C
লক্ষ্য করলে দেখা যায় যে B সেটের 1 উপাদান C সেটের উপাদান নয়।
সুতরাং B\C সেটের উপাদান সমূহ হবে: 1
অতএব, B\C = {1,2,a}\ {2,a,b} [ উভয় সেটের অন্তর ]
= {1}
নির্ণেয় B\C = 1

২.১.৩। A = {2,3,4}, B= {1,2,a} এবং C= {2,a,b} হলে, সেটগুলো নির্ণয় কর।
(খ) A∪B

সমাধান:
দেওয়া আছে,
A = {2,3,4}
B= {1,2,a}
C= {2,a,b}

A∪B - এর অৰ্থ হল, A সেট এবং B সেটের সমস্ত উপাদান নিয়ে গঠিত সেট।
এখানে লক্ষ্য রাখতে হবে কোন উপাদান যেন পুনরাবৃত্তি না হয়।
অর্থাৎ A∪B = A + B
লক্ষ্য করলে দেখা যায় যে A সেট এবং B সেটের সম্মিলিত উপাদান সমূহ হল: {1,2,3,4,a}
সুতরাং A∪B সেটের উপাদান সমূহ হবে: {1,2,3,4,a}
অতএব, A∪B = {2,3,4}∪ {1,2,a} [ উভয় সেটের সম্মিলিত উপাদান সমূহ ]
= {1,2,3,4,a}
নির্ণেয় A∪B = {1,2,3,4,a}

২.১.৩। A = {2,3,4}, B= {1,2,a} এবং C= {2,a,b} হলে, সেটগুলো নির্ণয় কর।
(গ) A∩C

সমাধান:
দেওয়া আছে,
A = {2,3,4}
B= {1,2,a}
C= {2,a,b}

A∩C - এর অৰ্থ হল, যে সমস্ত উপাদান সেট A এবং C সেট উভয়ই সেটা আছে। সেই সমস্ত উপাদান নিয়ে গঠিত সেট।
অর্থাৎ সেট A এবং C সেটের সাধারণ উপাদান নিয়ে গঠিত সেট।
লক্ষ্য করলে দেখা যায় যে A সেট এবং C সেটের সাধারণ উপাদান হল: {2}
সুতরাং A∩C সেটের উপাদান সমূহ হবে: {2}
অতএব, A∩C = {2,3,4} ∩ {2,a,b} [ উভয় সেটের সাধারণ উপাদান]
= {2}
নির্ণেয় A∩C = {2}

২.১.৩। A = {2,3,4}, B= {1,2,a} এবং C= {2,a,b} হলে, সেটগুলো নির্ণয় কর।
(ঘ) A∪(B∩C)

সমাধান:

A = {2,3,4}
B= {1,2,a}
C= {2,a,b}

(B∩C) = {1,2,a}∩ {2,a,b} [ উভয় সেটের সাধারণ উপাদান ]
= {2,a}
এখন, A∪(B∩C) = {2,3,4} ∪ {2,a} [ উভয় সেটের সম্মিলিত উপাদান সমূহ ]
= {2,3,4,a}
নির্ণেয় A∪(B∩C) = {2,3,4,a}

২.১.৩। A = {2,3,4}, B= {1,2,a} এবং C= {2,a,b} হলে, সেটগুলো নির্ণয় কর।
(ঙ) A∩(B∪C)

সমাধান:

= {1,2,a,b}
এখন, A∩(B∪C) = {2,3,4} ∩ {1,2,a,b} [ উভয় সেটের সাধারণ উপাদান]
= {2}
নির্ণেয় A∩(B∪C) = {2}
