২.১.৫। Q = {x,y} এবং R = {m,n,l} হলে, P(Q) এবং P(R)নির্ণয় কর।
 সমাধান:
দেওয়া আছে,
Q = {x,y}
R = {m,n,l}

P (Q)= {Q এর সম্ভাব্য উপসেট সমূহ }
= {{x},{y},{x,y},{ϕ}}

 P (R)= { R এর সম্ভাব্য উপসেট সমূহ }
= {{m},{n},{l},{m,n},{m,l},{n,l},{m,n,l},{ϕ}}

 P (Q) = {{x},{y},{x,y},{ϕ}
P (R) = {{m},{n},{l},{m,n},{m,l},{n,l},{m,n,l},{ϕ}}
