২.১.৬। A = {a,b}, B = {a,b,c} এবং C = A ∪ B হলে, দেখাও যে, P(C) এর উপাদান সংখ্যা $ 2^n $ যেখানে n হচ্ছে C এর উপাদান সংখ্যা।
 সমাধান:
দেওয়া আছে,
A = {a,b}
B = {a,b,c}

C = A ∪ B = উভয় সেটের সম্মিলিত উপাদান সমূহ
= {a,b} ∪ {a,b,c}
= {a,b,c}

 এখানে, C এর উপাদান সংখ্যা n = 3

P (C)= { C এর সম্ভাব্য উপসেট সমূহ }
= {{a},{b},{c},{a,b},{b,c},{c,a},{a,b,c},{ϕ}}

এখানে, P(C) এর উপাদান সংখ্যা = 8
= $2^3$
= $2^n$
যেখানে n হচ্ছে C এর উপাদান সংখ্যা।
