২.১.৮। (ক) P = {a}, Q ={b,c} হলে, $ P× Q $ এবং $ Q × P $ নির্ণয় কর।
(খ) A = {3,4,5}, B ={4,5,6} এবং C = {x,y}হলে, $ A ∩ B ×C $ নির্ণয় কর।

২.১.৮। (ক) P = {a}, Q ={b,c} হলে, $ P× Q $ এবং $ Q × P $ নির্ণয় কর।
সমাধান:
দেওয়া আছে,
P = {a}
Q ={b,c}

$ P× Q $ = P এবং Q এর কার্তেসীয় গুণজ সেট
= {a}$×${b,c}
= {(a,b),(a,c)}
এবং
$ Q× P $ = Q এবং P এর কার্তেসীয় গুণজ সেট
= {b,c}$×${a}
= {(b,a),(c,a)}

নির্ণেয়:
$ P× Q $ = {(a,b),(a,c)}
$ Q× P $ = {(b,a),(c,a)}

২.১.৮। (খ) A = {3,4,5}, B ={4,5,6} এবং C = {x,y}হলে, $ A ∩ B ×C $ নির্ণয় কর।
সমাধান:
দেওয়া আছে,
A = {3,4,5}
B ={4,5,6}
C = {x,y}

A ∩ B = উভয় সেটের সাধারণ উপাদান
={3,4,5} ∩ {4,5,6}
= {4,5}
এবং
(A ∩ B) $×$ C= (A ∩ B) এবং C এর কার্তেসীয় গুণজ সেট
= {4,5}$×${x,y}
= {(4,x),(4,y),(5,x),(5,y)}

নির্ণেয়:
(A ∩ B) $×$ C= {(4,x),(4,y),(5,x),(5,y)}

২.১.৮। (গ) P = {3,5,7}, Q ={5,7} এবং R = P\Q হলে, $ P ∪ Q ×R $ নির্ণয় কর।
সমাধান:
দেওয়া আছে,
P = {3,5,7}
Q ={5,7}
R = P\Q

P ∪ Q = উভয় সেটের সম্মিলিত উপাদান সমূহ
={3,5,7} ∪ {5,7}
= {3,5,7}
এবং
R = P\Q = উভয় সেটের অন্তর
= {3,5,7} - {5,7}
{3} (P ∪ Q) $×$ R= (P ∪ Q) এবং R এর কার্তেসীয় গুণজ সেট
= {3,5,7}$×${3}
= {(3,3),(5,3),(7,3)}

নির্ণেয়:
(P ∪ Q) $×$ R = {(3,3),(5,3),(7,3)}
