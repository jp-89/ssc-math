১। নিচের সেট গুলোকে তালিকা পদ্ধতিতে প্রকাশ কর।
ক) { $ x ∈ N:x^2 > 9 \text"এবং " x^3 < 130$ }
সমাধান:
এখানে , স্বাভাবিক সংখ্যার সেট N = { 1,2,3 ,4,5,.......}
প্রশ্নটির সমাধানের জন্য x বিভিন্ন মানের জন্য $x^2 $এবং $x ^3 $ মান নিম্নরূপ বের করি।
  x    $x^2$     $x^3$ 
  1    1    1 
  2    4    8 
  3    9    27 
  4    16    64 
  5    25    125 
  6    36    216 

উপরের তালিকা থেকে আমরা পাই ,
$ x^2 > 9 $ যখন x ≥ 4
$ x^3 < 130 $ যখন x ≤ 5
শর্তানুসারে , গ্রহণযোগ্য সংখ্যা সমূহ হল: 4 এবং 5
∴ নির্ণেয় সেট= {4, 5 } ( উত্তর )

১। নিচের সেট গুলোকে তালিকা পদ্ধতিতে প্রকাশ কর।
খ) { $ x ∈ Z:x^2 > 5 \text"এবং " x^3 ≤ 36$ } }
সমাধান:
এখানে , পূর্ণ সংখ্যার সেট Z = { ....,-3,-2, -1, 0, 1,2,3 ,4,5,.......}
প্রশ্নটির সমাধানের জন্য x বিভিন্ন মানের জন্য $x^2 $এবং $x ^3 $ মান নিম্নরূপ বের করি।
  x    $x^2$     $x^3$ 
  0    0    0 
  -1    1    -1 
  1    1    1 
  -2    4    -8 
  2    4    8 
  -3    9    -27 
  3    9    27 
  -4    16    -64 
  4    16    64 
  -5    25    -125 
  5    25    125 
  -6    36    -216 
  6    36    216 

উপরের তালিকা থেকে আমরা পাই ,
$ x^2 > 5 $ যখন x = ...,-5,-4,-3, 3,4,5,6
$ x^3 ≤ 36 $ যখন x =....,-5,-4,-3, 3
শর্তানুসারে , গ্রহণযোগ্য সংখ্যা সমূহ হল: ...-5,-4, -3, এবং 3
∴ নির্ণেয় সেট= { ...-5,-4, -3,3 } ( উত্তর )

১। নিচের সেট গুলোকে তালিকা পদ্ধতিতে প্রকাশ কর।
গ) { $ x ∈ N:x, 36 \text"এর গুণনীয়ক এবং 6 এর গুণিতক " $ }
সমাধান:
এখানে , স্বাভাবিক সংখ্যার সেট N = { 1,2,3 ,4,5,.......}
আমরা কে নিম্নরূপ গুণনীয়ক আকারে লিখতে পারি, = $ 1 ×36 $
= $ 2 ×18 $
= $ 3 ×12 $
= $ 4 ×9 $
= $ 6 ×6 $
অতএব , 36 এর গুণনীয়ক সমূহ {1, 2, 3, 4, 6, 9, 12, 18, 36}
আবার 6 এর গুণিতক সমূহকে নিম্ন আকারে প্রকাশ করা যায়
6 এর গুণিতক সমূহ = { 6, 12, 18, 24, 30, 36,42,48, 54, ...... }
যে সমস্ত সংখ্যা সমূহ উভয় গ্রূপের মধ্যে আছে সে গুলো হল,
= { 6, 12, 18, 36}
∴ শর্তানুসারে , গ্রহণযোগ্য নির্ণেয় সেট={ 6, 12, 18, 36} ( উত্তর )

১। নিচের সেট গুলোকে তালিকা পদ্ধতিতে প্রকাশ কর।
