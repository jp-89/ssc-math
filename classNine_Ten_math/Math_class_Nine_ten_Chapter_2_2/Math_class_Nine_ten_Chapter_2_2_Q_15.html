২.২.১৫। যদি $ f(x)= {2x + 1}/{2x-1} $ হয়, তবে $ {f(1/x^2)+1}/{ f(1/x^2)-1 } $ এর মান নির্ণয় কর

সমাধান:
দেওয়া আছে,
$ f(x)= {2x + 1}/{2x-1} $
অতএব,
$ f(1/x^2)= {2/x^2 + 1}/{2/x^2-1} $

= $ {{2+x^2}/x^2 }/{{2-x^2}/x^2} $

= $ {2+x^2}/{2-x^2} $

সুতরাং
$ {f(1/x^2)+1}/{ f(1/x^2)-1 } = { {2+x^2}/{2-x^2} +1 }/ { {2+x^2}/{2-x^2} -1 } $

=$ { {2+x^2 + 2-x^2 }/{2-x^2} }/ { {2+x^2 -2+x^2}/{2-x^2} } $

=$ {2+x^2 + 2-x^2 }/ {2+x^2 -2+x^2} $

=$ {4 }/ {2x^2} $

=$ {2 }/ {x^2} $

∴ নির্ণেয় মান = $ {2 }/ {x^2} $
