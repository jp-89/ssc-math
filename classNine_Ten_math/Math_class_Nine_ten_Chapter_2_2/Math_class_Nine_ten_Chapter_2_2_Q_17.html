২.২.১৭। নিচের অম্বয়গুলো থেকে ডোমেন এবং রেঞ্জ নির্ণয় কর।
(ক) $ R = \{(2,1),(2,2),(2,3)\} $

সমাধান:
দেওয়া আছে,
$ R = \{(2,1),(2,2),(2,3)\} $
আমরা জানি, কোনো অম্বয়ের ক্রোমজোড়গুলোর প্রথম উপাদানসমূহের সেটকে এর ডোমেন এবং দ্বিতীয় উপাদানসমূহের সেটকে এর রেঞ্জ বলে।

সুতরাং,অম্বয়ে ক্রোমজোড়গুলোর প্রথম উপাদানসমূহ = 2,2,2 এবং
দ্বিতীয় উপাদানসমূহ = 1,2,3
অতএব, ডোমেন বা ডোম = { 2,2,2 }
এবং রেঞ্জ = { 1,2,3}

ডোমেন এবং রেঞ্জ এর উপাদানসমূহ যদি পুনরাবৃত হয় তবে তাদের একটি মান গ্ৰহন করা হয়।
এখানে দেখা যায় যে ডোমেন এর ক্ষেত্রে 2 পুনরাবৃত হচ্ছে।
সুতরাং এখানে ,ডোমেন বা ডোম = { 2 }
এবং রেঞ্জ = { 1,2,3}

নির্ণেয়, ডোমেন বা ডোম = { 2 }
এবং রেঞ্জ = { 1,2,3}


২.২.১৭। নিচের অম্বয়গুলো থেকে ডোমেন এবং রেঞ্জ নির্ণয় কর।
