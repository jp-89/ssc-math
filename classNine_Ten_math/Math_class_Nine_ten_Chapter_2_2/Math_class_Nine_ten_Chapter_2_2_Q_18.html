২.২.১৮। নিচের অম্বয়গুলো তালিকা পদ্ধতিতে প্রকাশ কর এবং ডোমেন ও রেঞ্জ নির্ণয় কর।
(ক) $ R = \{(x,y):x∈A, y∈A $ এবং x + y =1}, যেখানে A = {-2,-1,0,1,2 }

সমাধান:
দেওয়া আছে,
$ R = \{(x,y):x∈A, y∈A $ এবং x + y =1}, , যেখানে A = {-2,-1,0,1,2 }

এখানে দেখা যায় যে x এবং y উভয়ই A এর উপসেট এবং
R এর শর্তাবলীর ক্ষেত্রে , x ও y এর মধ্যে সম্পর্ক হল:
x + y = 1
 বা, y = 1-x

এখন আমরা x এর বিভিন্ন মানের জন্য y এর মান সমূহ বের করি।
 x     y = 1-x  
 -2   3  
  -1  2  
 0   1  
 1   0  
 2   -1  

উপরোক্ত তালিকা হতে দেখা যায় যে,
  • y = 3, A এর উপসেট নয়।
  • অথবা 3 ∉ A অর্থাৎ y = 3 গ্রহণযোগ্য নয়।
  • যেহেতু y = 3 গ্রহণযোগ্য নয়, সেহেতু x = - 2 গ্রহণযোগ্য নয়।
  • সুতরাং (-2,3) ∉ R
  • অর্থাৎ অম্বয় R এর গ্রহণযোগ্য মানসমূহ হল,
    (-1,2),(0,1), (1,0),(2-1)

অতএব, R= {(-1,2),(0,1), (1,0),(2-1)}
 আমরা জানি, কোনো অম্বয়ের ক্রোমজোড়গুলোর প্রথম উপাদানসমূহের সেটকে এর ডোমেন এবং দ্বিতীয় উপাদানসমূহের সেটকে এর রেঞ্জ বলে।

সুতরাং,অম্বয়ে ক্রোমজোড়গুলোর প্রথম উপাদানসমূহ = -1,0,1,2 এবং
দ্বিতীয় উপাদানসমূহ = 2,1,0,-1
অতএব, ডোমেন বা ডোম = { -1,0,1,2 }
এবং রেঞ্জ = { 2,1,0,-1]



নির্ণেয়,
R = {(-1,2),(0,1), (1,0),(2-1)}
ডোমেন বা ডোম = { -1,0,1,2 }
এবং রেঞ্জ = { 2,1,0,-1]

২.২.১৮। নিচের অম্বয়গুলো তালিকা পদ্ধতিতে প্রকাশ কর এবং ডোমেন ও রেঞ্জ নির্ণয় কর।
(খ) $ F = \{(x,y):x∈C, y∈C $ এবং y =2x}, যেখানে C = {-1,0,1,2, 3 }

সমাধান:
দেওয়া আছে,
$ F = \{(x,y):x∈C, y∈C $ এবং y =2x}, যেখানে C = {-1,0,1,2, 3 }

এখানে দেখা যায় যে x এবং y উভয়ই C এর উপসেট এবং
R এর শর্তাবলীর ক্ষেত্রে , x ও y এর মধ্যে সম্পর্ক হল:
y = 2x
 বা, $ y =2x $

এখন আমরা x এর বিভিন্ন মানের জন্য y এর মান সমূহ বের করি।
 x     $y = 2x $  
 -1   -2  
  0  0  
 1   2  
 2   4  
 3   6  

উপরোক্ত তালিকা হতে দেখা যায় যে,
  • y = -2,4,6 C এর উপসেট নয়।
  • অথবা -2,4,6 ∉ C অর্থাৎ y = -2,4,6 গ্রহণযোগ্য নয়।
  • যেহেতু y = -2,4,6 গ্রহণযোগ্য নয়, সেহেতু x = - 1,2,3 গ্রহণযোগ্য নয়।
  • সুতরাং (-1,-2) , (2,4), (3,6) ∉ F
  • অর্থাৎ অম্বয় F এর গ্রহণযোগ্য মানসমূহ হল,
    (0,0),(1,2)

অতএব, F= {(0,0),(1,2)}
 আমরা জানি, কোনো অম্বয়ের ক্রোমজোড়গুলোর প্রথম উপাদানসমূহের সেটকে এর ডোমেন এবং দ্বিতীয় উপাদানসমূহের সেটকে এর রেঞ্জ বলে।

দ্বিতীয় উপাদানসমূহ =0,2
