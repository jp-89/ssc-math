২.২.২। সেট C হতে সেট B এ একটি সম্পর্ক R হলে নিচের কোনটি সঠিক?
(ক) $ R ⊂ C $
(খ) $ R ⊂ B $
(গ) $ R ⊆ C ×B $
(ঘ) $C ×B ⊆ R$

সমাধান:
আমরা জানি, সেট C হতে সেট B এ একটি সম্পর্ক R হলে R কে নিম্নোক্ত আকারে প্রকাশ করা হয়।
$ R ⊆ C ×B $
এখানে R সেট $ R ⊆ C ×B $ সেটের একটি উপসেট এবং উপসেট R কে সেট C হতে B সেটের একটি অম্বয় বলা হয়।
সুতরাং, সেট C হতে সেট B এ একটি সম্পর্ক R হল,
$ R ⊆ C ×B $ (গ)

ব্যাখ্যা :
(ক) $ R ⊂ C $
$ R ⊂ C $ হতে পারে না কারন উপসেট R শুধুমাত্র সেট C এর উপসেটব নয়। উপসেট R হল C এবং B সেটদ্বয়ের কার্তেসীয় গুনজ $C ×B $ সেটের উপসেট।

(খ) $ R ⊂ B $
$ R ⊂ B $ হতে পারে না কারন উপসেট R শুধুমাত্র সেট B এর উপসেটব নয়। উপসেট R হল C এবং B সেটদ্বয়ের কার্তেসীয় গুনজ $C ×B $ সেটের উপসেট।

(গ) $ R ⊆ C ×B $
$ R ⊆ C ×B $ সত্য কারন উপসেট R হল C এবং B সেটদ্বয়ের কার্তেসীয় গুনজ $C ×B $ সেটের উপসেট।

(ঘ) $C ×B ⊆ R$
$C ×B ⊆ R$ হতে পারে না কারন $C ×B $, R এর উপসেট নয় বরং উপসেট R হল C এবং B সেটদ্বয়ের কার্তেসীয় গুনজ $C ×B $ সেটের উপসেট।
