২.২.২১। সার্বিক সেট U ={ x :x ∈ N এবং বেজোড় সংখ্যা }
A = { x ∈ N : 2 ≤ x ≤ 7}
B = { x ∈ N : 3 < x < 6}
C = { x ∈ N : $ x^2 >5 $ এবং $ x^3 <130 $ }
(ক) A সেটকে তালিকা পদ্ধতিতে প্রকাশ কর।
(খ) $ A^' $ এবং C-B নিৰ্ণয় কর।
(গ) $ B×C $ এবং P(A ∩ C) নির্ণয় কর।

সমাধান:
যেহেতু সার্বিক সেট U যেকোন স্বাভাবিক সংখ্যা এবং বেজোড় সংখ্যা সেহেতু U কে নিম্নোক্ত আকারে প্রকাশ করা যায়।
$U = \{1,3,5,7,9,11,.......\}$
(ক) A সেটকে তালিকা পদ্ধতিতে প্রকাশ কর।
দেওয়া আছে,
A = $ \{ x ∈ N : 2 ≤ x ≤ 7\}$
অতএব,
A = {2,3,4,5,6,7}

নির্ণেয়:
A = {2,3,4,5,6,7}

(খ) $ A^' $ এবং C-B নিৰ্ণয় কর।
সমাধান:
দেওয়া আছে,
B = { x ∈ N : 3 < x < 6}
C = { x ∈ N : $ x^2 >5 $ এবং $ x^3 <130 $ }
উপরের (ক) এর সমাধান থেকে পাই,
$U = \{1,3,5,7,9,11,.......\}$
$A = \{2,3,4,5,6,7\}$
অতএব,
$A^'$ = সার্বিক সেট - $ A $
= U -A
= $\{1,3,5,7,9,11,.......\}- \{2,3,4,5,6,7 \}$
 = $ \{1,9,11,.......\} $
দেওয়া আছে,
B = { x ∈ N : 3 < x < 6}
অতএব,
B = {4,5}

আবার দেওয়া আছে,
C = { x ∈ N : $ x^2 >5 $ এবং $ x^3 <130 $ }
এখানে , স্বাভাবিক সংখ্যার সেট N = { 1,2,3 ,4,5,.......}
প্রশ্নটির সমাধানের জন্য x বিভিন্ন মানের জন্য $x^2 $এবং $x ^3 $ মান নিম্নরূপ বের করি।
  x    $x^2$     $x^3$ 
  1    1    1 
  2    4    8 
  3    9    27 
  4    16    64 
  5    25    125 
  6    36    216 

উপরের তালিকা থেকে আমরা পাই ,
$ x^2 > 5 $ যখন x ≥ 3
$ x^3 < 130 $ যখন x ≤ 5
শর্তানুসারে , গ্রহণযোগ্য সংখ্যা সমূহ হল: 3, 4 এবং 5
∴ C = {3, 4, 5 }

C \B অথবা C-B = উভয় সেটের অন্তর
= {3, 4, 5 } - {4,5}
= {3}

নির্ণেয়:
$A^'$ = = $ \{1,9,11,.......\} $
C-B = $\{3\}$

(গ) $ B×C $ এবং P(A ∩ C) নির্ণয় কর।
সমাধান:
উপরের (ক) এবং (খ )এর সমাধান থেকে পাই,
$U = \{1,3,5,7,9,11,.......\}$
$A = \{2,3,4,5,6,7\}$
$B = \{4,5\}$
$C = \{3, 4, 5 \}$
অতএব,
$ B×C $ = B এবং C এর কার্তেসীয় গুণজ সেট
= $\{4,5\}×\{3, 4, 5 \}$
= {(4,3),(4,4),(4,5),(5,3),(5,4),(5,5)}


$ A ∩ C $ = উভয় সেটের সাধারণ উপাদান
= $\{2,3,4,5,6,7\} ∩ \{3, 4, 5 \}$
= {(3,4,5)}


$ P(A ∩ C ) $ = { (A ∩ C ) এর সম্ভাব্য উপসেট সমূহ }
= {{3}, {4}, {5},{3,4},{3,5},{4,5},{3,4,5},{ϕ} }
= {{3}, {4}, {5},{3,4},{3,5},{4,5},{3,4,5},{ϕ} }


নির্ণেয়:
$ B×C $ = {(4,3),(4,4),(4,5),(5,3),(5,4),(5,5)}
$ P(A ∩ C ) $ = {{3}, {4}, {5},{3,4},{3,5},{4,5},{3,4,5},{ϕ} }
