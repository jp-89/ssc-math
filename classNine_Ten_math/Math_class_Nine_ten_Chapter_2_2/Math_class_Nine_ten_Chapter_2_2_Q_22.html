২.২.২২ -খ) উদ্দীপক ব্যবহার করে,
$A ∪(B ∩ C ) = (A ∪B ) ∩ (A ∪ C ) $
সম্পর্কটির সত্যতা যাচাই কর।
B∩C - এর অৰ্থ হল, যে সমস্ত উপাদান সেট B এবং C সেট উভয়ই সেটে আছে, সেই সমস্ত উপাদান নিয়ে গঠিত সেট।
অর্থাৎ সেট B এবং C সেটের সাধারণ উপাদান নিয়ে গঠিত সেট।
লক্ষ্য করলে দেখা যায় যে B সেট এবং C সেটের সাধারণ উপাদান হল: $\{3,5\} $
 সুতরাং, $ B∩C = \{3,5 \}$

আবার,প্রদত্ত ভেন চিত্রে লক্ষ্য করলে দেখা যায় যে,
$A = \{1,2,3,4\}$

অতএব,
বামপক্ষ = $A ∪(B ∪C )$
= $ \{1,2,3,4\} ∪ \{3,5\}$
= $ \{1,2,3,4,5\} $ উভয় সেটের সম্মিলিত উপাদান সমূহ

$A ∪ C = \{1,2,3,4,5,6\} $

ডানপক্ষ= $(A ∪B ) ∩ (A ∪ C ) $
= $ \{1,2,3,4,5,7\} ∩ \{1,2,3,4,5,6\} $
= $\{1,2,3,4,5\}$ উভয় সেটের সাধারণ উপাদান
