২.২.২৩। $y = f(x) = { 4x-7}/{2x-4}$ একটি ফাংশন।
ক) $f( - 1/2) $ এর মান নির্ণয় কর।
খ) $ { f(x) +2 }/{ f(x) -1} $ এর মান নির্ণয় কর।
গ) দেখাও যে, $ f(y) = x $

২.২.২৩ - ক) $f( - 1/2) $ এর মান নির্ণয় কর।
সমাধান:
দেওয়া আছে,
$ f(x) = { 4x-7}/{2x-4}$
 অতএব,
$ f(- 1/2 ) = { 4 (- 1/2) -7}/{2(- 1/2)-4}$

= $ { -2 -7}/{-1-4}$

= $ { -9}/{-5}$

= $ { 9}/{5}$

২.২.২৩ - খ) $ { f(x) +2 }/{ f(x) -1} $ এর মান নির্ণয় কর।
 সমাধান:
দেওয়া আছে,
$ { f(x) +2 }/{ f(x) -1} $

 $f(x) $ এর মান বসিয়ে পাই,
=$ { { 4x-7}/{2x-4} +2 }/{ { 4x-7}/{2x-4} -1} $

= $ { { 4x-7 +4x-8}/{2x-4} }/{ { 4x-7-2x+4}/{2x-4} } $

= $ { { 4x-7 +4x-8} }/{ { 4x-7-2x+4} } $

= $ { {8x-15} }/{ { 2x-3} } $

= $ { {8x-15} }/{ { 2x-3} } $



২.২.২৩ -গ) দেখাও যে, $ f(y) = x $
 সমাধান:
দেওয়া আছে,
$ f(x) = { 4x-7}/{2x-4}$

অতএব,
$ f(y) = { 4y-7}/{2y-4}$

আবার, আমরা পাই,
$y = { 4x-7}/{2x-4}$

বা, $ 2xy -4y = 4x-7 $
বা, $ 2xy -4x = 4y-7 $
বা, $ x(2y -4) = 4y-7 $
বা, $ x = {4y-7}/{ 2y -4} $

বা, $ x = f(y) $
বা, $ f(y) = x $ (প্রমাণিত)
