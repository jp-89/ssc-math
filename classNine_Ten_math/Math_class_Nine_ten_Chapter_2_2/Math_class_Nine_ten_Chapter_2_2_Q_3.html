২.২.৩। যদি $A = \{1,2\}, B = \{2,4\} $ হলে, P(A ∩ B ) এর সদস্য সংখ্যা নিচের কোনটি?
ক) 1
খ) 2
গ) 3
ঘ) 8

সমাধান:
দেওয়া আছে,
$A = \{1,2\} $ এবং
$ B = \{2,4\} $

সুতরাং,
$(A ∩ B ) = \{1,2\} ∩ \{2,4\} $
= $\{2\} $ [ উভয় সেটের সাধারণ উপাদান। ]

P(A ∩ B ) এর সদস্য সংখ্যা = $2^n$
=$2^1 $ (এখানে, $n = (A ∩ B)$ সেটের উপাদান সংখ্যা।)
=$2$

অতএব, সঠিক উত্তর: খ) 2
