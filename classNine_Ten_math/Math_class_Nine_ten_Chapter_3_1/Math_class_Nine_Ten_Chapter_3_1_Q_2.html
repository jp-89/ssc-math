৩.১.২(ক) সরল কর:
$(2a + 7)^2+2(2a + 7)(2a -7 )+(2a + 7)^2$
সমাধান:
প্রদত্ত রাশী
= $(2a + 7)^2+2(2a + 7)(2a -7 )+(2a + 7)^2$

ধরি,
$ 2a + 7 = x $ এবং
$ 2a - 7 = y $
সুতরাং, আমরা পাই,
$x^2+2xy+y^2$
=$(x+y)^2 $

মান বসিয়ে পাই,
=$(2a + 7+2a - 7)^2 $
=$(4a)^2 $
= $16a^2 $

নির্ণেয় মান :
$16a^2 $

৩.১.২(খ) সরল কর:
$(3x+2y)^2+2(3x+2y)(3x-2y )+(3x-2y)^2$

= $(3x+2y)^2+2(3x+2y)(3x-2y )+(3x-2y)^2$

ধরি,
$ 3x+2y = a $ এবং
$ 3x+2y = b $
সুতরাং, আমরা পাই,
$a^2+2ab+b^2$
=$(a+b)^2 $

মান বসিয়ে পাই,
=$(3x+2y+3x-2y)^2 $
=$(6x)^2 $
= $36x^2 $

নির্ণেয় মান :
$36x^2 $

৩.১.২(গ) সরল কর:
$(7p+3r-5x)^2-2(7p+3r-5x)(8p-43r-5x )+(8p-43r-5x)^2$
সমাধান:
প্রদত্ত রাশী
= $(7p+3r-5x)^2-2(7p+3r-5x)(8p-43r-5x )+(8p-43r-5x)^2$

ধরি,
$ 7p+3r-5x = a $ এবং
$ 8p-43r-5x = b $
সুতরাং, আমরা পাই,
$a^2-2ab+b^2$
=$(a-b)^2 $

মান বসিয়ে পাই,
=$\{(7p+3r-5x)-(8p-4r-5x)\}^2 $
=$(7p+3r-5x-8p+4r+5x)^2 $
=$(-p+7r)^2 $
=$(7r-p)^2 $
=$(7r)^2-2(7r)(p)+(p)^2 $
=$49r^2-14pr+p^2 $
=$p^2-14pr+49r^2 $

নির্ণেয় মান :
$p^2-14pr+49r^2 $

৩.১.২(ঘ) সরল কর:
$(2m+3n-p)^2+ (2m-3n+p)^2-2(2m+2n-p)(2m-3n+p )$
সমাধান:
প্রদত্ত রাশী
= $(2m+3n-p)^2+ (2m-3n+p)^2-2(2m+3n-p)(2m-3n+p )$

ধরি,
$ 2m+3n-p = a $ এবং
$ 2m-3n+p = b $
সুতরাং, আমরা পাই,
$a^2+b^2-2ab$
$a^2-2ab+b^2$
=$(a-b)^2 $

মান বসিয়ে পাই,
=$\{(2m+3n-p)-(2m-3n+p)\}^2 $
=$(2m+3n-p-2m+3n-p)^2 $
=$(6n-2p)^2 $
=$(6n)^2-2(6n)(2p)+(2p)^2 $
=$36n^2-24pn+4p^2 $

নির্ণেয় মান :
$36n^2-24pn+4p^2 $

= $6.35×6.35+2×6.35×3.65+3.65×3.65$

$ 6.35 = a $ এবং

৩.১.২(ছ ) সরল কর:

= ${7529×7529-7519×7519}/{7529+7519}$

ধরি,

$a-b $

=$(7529-7519) $
