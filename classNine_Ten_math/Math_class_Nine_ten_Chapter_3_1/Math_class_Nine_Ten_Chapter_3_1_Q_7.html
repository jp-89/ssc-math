৩.১.৭। $x-1/x = 4$ হলে, প্রমান কর যে $x^4-1/x^4 = 322$?
সমাধান:
দেওয়া আছে,
$ x-1/x = 4$
উভয় পক্ষকে বর্গ করিয়া পাই,
$(x-1/x)^2 = 4^2$
বা, $(x-1/x)^2 = 4^2$
বা, $x^2+1/x^2-2x1/x = 16$
বা, $x^2+1/x^2-2 = 16$
বা, $x^2+1/x^2 = 16 + 2$
বা, $x^2+1/x^2 = 18$
উভয় পক্ষকে আবার বর্গ করিয়া পাই,
$(x^2+1/x^2)^2 = (18)^2$
বা, $(x^2)^2+(1/x^2)^2+2x^2{1/x^2} = 324$
বা, $x^4+1/x^4+2 = 324$
বা, $x^4+1/x^4 = 324-2$
বা, $x^4+1/x^4= 322$
$x^4+1/x^4 = 322$ (প্রমাণিত )
