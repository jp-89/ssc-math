৩.২.১৫.(ক)। যদি a+b+c = 0 হয়, তবে দেখাও যে ,
$a^3+b^3+c^3 = 3bc$
সমাধান:
দেওয়া আছে,
$ a+b+c = 0 $
বা, a+b = -c
উভয় পক্ষকে ঘন করিয়া পাই,
$(a+b)^3 =(-c)^3 $
বা, $a^3+b^3 +3(ab )(a+b)= -c^3 $

a+b = -c মান বসিয়ে পাই,
বা, $a^3+b^3 +3(ab )(-c)= -c^3 $

বা, $a^3+b^3 -3abc= -c^3 $

বা, $a^3+b^3 +c^3= 3abc $

$a^3+b^3 +c^3= 3abc $(প্রমানিত )

৩.২.১৫.(খ)। যদি a+b+c = 0 হয়, তবে দেখাও যে ,
$ {(b+c)^2}/{3bc }+ {(c+a)^2}/{3ca } +{(a+b)^2}/{3ab } =1 $

সমাধান:
দেওয়া আছে,
a+b+c = 0
a+b = -c .....(1)
বা, a+c = -b .....(2)
বা, b+c = -a .....(3)
আবার দেওয়া আছে,
$ {(b+c)^2}/{3bc }+ {(c+a)^2}/{3ca } +{(a+b)^2}/{3ab } $
উপরে প্রাপ্ত মান সমূহ বসিয়ে পাই,
বা,$ {(-a)^2}/{3bc }+ {(-b)^2}/{3ca } +{(-c)^2}/{3ab } $
বা, $ {a^2}/{3bc }+ {b^2}/{3ca } +{c^2}/{3ab } $
বা, $ {a^3+b^3+c^3}/{3abc } $
উপরে (ক) থেকে $a^3+b^3 +c^3= 3abc $ প্রাপ্ত মান সমূহ বসিয়ে পাই,
= $ {{3abc}/{3abc } $
= 1
