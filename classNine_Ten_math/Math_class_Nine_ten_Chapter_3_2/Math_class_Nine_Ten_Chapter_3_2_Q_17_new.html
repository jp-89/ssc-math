৩.২.১৭। $x-1/x = √3 $ যেখানে $ x ≠ 0 $

ক) প্রমান কর যে, $x^2-√3x = 1$
খ) প্রমান কর যে, $23(x^2+1/x^2) = 5(x^4+1/x^4)$
গ) $x^6 + 1/x^6 $ এর মান নির্ণয় কর।
ক) প্রমান কর যে, $x^2-√3x = 1$
সমাধান:
দেওয়া আছে,
$x-1/x = √3 $
বা, ${x^2-1}/x = √3 $

বা, ${x^2-1} = √3x $
বা, $ x^2-√3x = 1 $ (প্রমাণিত)

খ) প্রমান কর যে, $23(x^2+1/x^2) = 5(x^4+1/x^4)$
সমাধান:
দেওয়া আছে,
$x-1/x = √3 $
উভয় পক্ষকে বর্গ করিয়া পাই,
$(x-1/x)^2 = (√3)^2 $
বা, $ x^2- 2. x. 1/x +1/x^2 = 3 $

বা, $ x^2- 2. +1/x^2 = 3 $

বা, $ x^2 +1/x^2 = 3+2 $

বা, $ x^2 +1/x^2 = 5 $ ....(1)

(1) সমীকরণের উভয় পক্ষকে বর্গ করিয়া পাই,
বা, $ ( x^2 +1/x^2 )^2= (5)^2 $
বা, $ (x^2)^2 + 2. x^2. 1/x^2 +(1/x^2)^2 = 25 $

বা, $ x^4 + 2 +1/x^4 = 25 $

বা, $ x^4 +1/x^4 = 25-2 $

বা, $ x^4 +1/x^4 =23 $ ....(2)

(1) কে 23 দ্বারা গুণ করিলে পাই,
$ 23(x^2 +1/x^2) = 5×23 $

বা, $ 23(x^2 +1/x^2) = 5(x^4 +1/x^4) $ (প্রমাণিত)


গ) $x^6 + 1/x^6 $ এর মান নির্ণয় কর।
সমাধান:
উপরের সমাধান থেকে পাই,
$ x^2 +1/x^2 = 5 $ ...(1)
$ x^4 +1/x^4 =23 $ ...(2)

সমীকরণদ্বয় গুণ করিলে পাই,
$(x^2 +1/x^2)( x^4 +1/x^4) = 5×23$

বা, $x^6 +1/x^2 + x^2 +1/x^6 = 115$

বা, $x^6 +1/x^6 + x^2 + +1/x^2 = 115$

বা, $x^6 +1/x^6 + 5 = 115$

বা, $x^6 +1/x^6 = 115-5$

বা, $x^6 +1/x^6 = 110$
