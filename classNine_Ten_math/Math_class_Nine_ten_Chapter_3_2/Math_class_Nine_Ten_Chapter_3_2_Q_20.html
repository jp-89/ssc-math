৩.২.২০। $a^4-a^2+1=0 $ হলে প্রমান কে যে,
$a^3+1/a^3 =0$

সমাধান:
দেওয়া আছে,
$a^4-a^2+1=0 $
বা,$a^4+1=a^2 $
বা,$a^2+1/a^2=1 $
এখন প্রদত্ত রাশী = $a^3+1/a^3$

= $(a+1/a)(a^2-a.1/a+1) $

= $(a+1/a)(a^2+1-a.1/a) $

= $(a+1/a)(a^2+1-1) $

$a^2+1/a^2=1 $ মান বসিয়ে পাই
= $(a+1/a)(1-1) $

= $(a+1/a)(0) $

= 0
অতএব,
$a^3+1/a^3 =0$(প্রমানিত )
