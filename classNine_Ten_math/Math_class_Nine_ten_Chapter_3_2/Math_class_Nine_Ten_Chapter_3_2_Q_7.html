৩.২.৭। যদি a+b = m, $a^2 +b^2= n $ এবং $ a^3+b^3 = p^3$ হয় তবে দেখাও যে $m^3+2p^3 = 3mn$
সমাধান:
দেওয়া আছে,
$a+b = m $
উভয় পক্ষকে বর্গ করিয়া পাই,
$(a+b)^2 = m^2 $
বা, $a^2+b^2+2ab = m^2 $
$a^2 +b^2= n $ মান বসিয়ে পাই,
$ n+2ab = m^2 $
$ ab = {m^2 -n }/2$

আবার,
$a+b = m $
উভয় পক্ষকে ঘন করিয়া পাই,
$(a+b)^3 = m^3 $
বা, $a^3+b^3+3ab(a+b) = m^3 $
মান বসিয়ে পাই,
বা, $p^3+3\{ {m^2 -n }/2\}(m) = m^3 $
বা, $p^3+ {3m^3 -3mn }/2 = m^3 $
বা, ${2p^3+ 3m^3 -3mn }/2 = m^3 $
বা, $2p^3+ 3m^3 -3mn = 2m^3 $
বা, $2p^3+ m^3 = 3mn $
বা, $ m^3 +2p^3 = 3mn $
$m^3 +2p^3 = 3mn $(প্রমানিত )
