৩.৩.১৩ । উৎপাদকে বিশ্লেষণ কর :
$2b^2c^2+2c^2a^2+2a^2b^2-a^4-b^4-c^4$
সমাধান:
দেওয়া আছে,
$2b^2c^2+2c^2a^2+2a^2b^2-a^4-b^4-c^4$
 =$2c^2(b^2+a^2)+4a^2b^2-2a^2b^2-a^4-b^4-c^4$
= $4a^2b^2-c^4+2c^2(a^2+b^2) -2a^2b^2-a^4-b^4$
= $4a^2b^2-c^4+2c^2(a^2+b^2) -a^4 -2a^2b^2-b^4$
= $4a^2b^2-\{c^4-2c^2(a^2+b^2)$ + $a^4 +2a^2b^2+b^4\}$
= $4a^2b^2-$$\{c^4-2c^2(a^2+b^2) +(a^2+b^2)^2\}$
= $4a^2b^2-$$\{(c^2)^2-2c^2(a^2+b^2) +(a^2+b^2)^2\}$
= $4a^2b^2-(c^2-a^2-b^2)^2$
= $(2ab)^2-(c^2-a^2-b^2)^2$
= $(2ab+c^2-a^2-b^2)(2ab-c^2+a^2+b^2)$
= $(c^2-a^2+ 2ab-b^2)($$a^2+ 2ab+b^2$$-c^2)$
= $\{c^2-(a^2- 2ab+b^2)\}\{(a+b)^2-c^2)\}$
= $\{c^2-(a-b)^2)\}(a+b+c)(a+b-c)$
= $(c+a-b)(c-a+b)(a+b+c)(a+b-c)$

নির্ণেয় উৎপাদক:
$(c+a-b)(c-a+b)(a+b+c)(a+b-c)$
