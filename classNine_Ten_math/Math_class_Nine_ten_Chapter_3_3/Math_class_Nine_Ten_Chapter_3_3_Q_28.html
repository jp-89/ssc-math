৩.৩.২৮। উৎপাদকে বিশ্লেষণ কর :
$(4a-3b)^2-2 (4a-3b)(a+2b)-35 (a+2b)^2$
সমাধান:
দেওয়া আছে,
$(4a-3b)^2-2 (4a-3b)(a+2b)-35 (a+2b)^2$
 ধরি,
4a-3b = x এবং
a+2b = y
অতএব, আমরা পাই,
$x^2-2xy-35y^2$
=$x^2-7xy+5xy-35y^2$
=$x (x-7y)+5y(x-7y) $
=$(x+5y)(x-7y) $

x ও y এর মান বসিয়ে পাই
=$(4a-3b+5a+10b)(4a-3b-7a-14b) $
=$(9a+7b)(-3a-17b) $
=$-(9a+7b)(3a+17b) $

নির্ণেয় উৎপাদক:
$-(9a+7b)(3a+17b) $
