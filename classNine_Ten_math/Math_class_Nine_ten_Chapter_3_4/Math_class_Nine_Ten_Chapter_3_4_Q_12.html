৩.৪.১২। উৎপাদকে বিশ্লেষণ কর:
$ x^3-3x^2+4x-4 $
সমাধান:
প্রদত্ত রাশি= $ x^3-3x^2+4x-4 $
এখানে x কে চলমান হিসাবে বিবেচনা করি।
প্রদত্ত রাশিকে x এর বহুপদী হিসাবে বিবেচনা করে,
ধরি,
$f(x) = x^3-3x^2+4x-4 $
যদি $x = 2$ হয়, তবে আমরা পাই,
$f(2) = (2)^3-3(2)^2+4(2)-4 $
বা, $f(2) =8-12+8-4 $
= 0

যেহেতু x = 2 হলে প্রদত্ত রাশিটির মান শূন্য হয়।
সেহেতু, $(x-2)$ উহার একটি উৎপাদক।
অতএব,আমরা পাই,
$f(x)= x^3-3x^2+4x-4 $
= $ x^2(x-2)-x(x-2)+2(x-2 ) $
= $ (x -2) (x^2-x +2) $
নির্ণেয় উৎপাদক = $(x -2) (x^2-x +2)$

সংক্ষেপে,
$ x^3-3x^2+4x-4 $
=$ x^3-2x^2-x^2+2x+2x-4 $
= $ x^2(x-2)-x(x-2)+2(x-2 ) $
= $ (x -2) (x^2-x +2) $
