৩.৪.১৪। উৎপাদকে বিশ্লেষণ কর:
$ x^3-x-24 $
সমাধান:
প্রদত্ত রাশি= $ x^3-x-24 $
এখানে x কে চলমান হিসাবে বিবেচনা করি।
প্রদত্ত রাশিকে x এর বহুপদী হিসাবে বিবেচনা করে,
ধরি,
$f(x) = x^3-x-24 $
যদি $x = 3$ হয়, তবে আমরা পাই,
$f(3) = (3)^3-3-24 $
বা, $f(3) =27-3-24 $
= 0

যেহেতু x = 3 হলে প্রদত্ত রাশিটির মান শূন্য হয়। সেহেতু, $(x-3)$ উহার একটি উৎপাদক।
অতএব,আমরা পাই,
$f(x)= x^3-x-24 $
= $ x^2(x-3)+3x(x-3)+8(x-3 ) $
= $ (x -3) (x^2+ 3x +8) $
নির্ণেয় উৎপাদক = $(x -3) (x^2+ 3x +8)$

সংক্ষেপে,
$ x^3-x-24 $
= $ x^3-3x^2+3x^2-9x+8x-24 $
= $ x^2(x-3)+3x(x-3)+8(x-3 ) $
= $ (x -3) (x^2+ 3x +8) $
