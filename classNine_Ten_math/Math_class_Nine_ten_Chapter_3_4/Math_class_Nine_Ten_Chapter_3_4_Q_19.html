৩.৪.১৯। উৎপাদকে বিশ্লেষণ কর:
$4x^3 -5x^2+5x-1 $
সমাধান:
প্রদত্ত রাশি= $4x^3 -5x^2+5x-1$
এখানে x কে চলমান হিসাবে বিবেচনা করি।
প্রদত্ত রাশিকে x এর বহুপদী হিসাবে বিবেচনা করে,
ধরি,
$f(x) = 4x^3 -5x^2+5x-1 $
যদি $x = 1/4$ হয়, তবে আমরা পাই,
$f(1/4) = 4(1/4)^3 -5(1/4)^2+5(1/4)-1 $
বা, $f(1/4) =4/{64} -5/{16} +5/4 -1 $
=$ 1/{16} -5/{16} -5/4 -1 $
=$ {1-5 +20 -16}/{16} $
= 0

যেহেতু x = 1/4 হলে প্রদত্ত রাশিটির মান শূন্য হয়। সেহেতু, $(x-1/4)$ বা, 4x-1 উহার একটি উৎপাদক।
অতএব,আমরা পাই,
$f(x)= 4x^3 -5x^2+5x-1$
= $ x^2( 4x-1)-x(4x-1)+1(4x-1) $
= $(4x-1)(x^2-x+1)) $
নির্ণেয় উৎপাদক = $(4x-1)(x^2-x+1)$

সংক্ষেপে,
$4x^3 -5x^2+5x-1$
= $4x^3 -x^2-4x^2+x+4x-1$
= $ x^2( 4x-1)-x(4x-1)+1(4x-1) $
= $(4x-1)(x^2-x+1) $
