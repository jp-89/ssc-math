৩.৪.২। উৎপাদকে বিশ্লেষণ কর:
$3a^3 +2a +5 $
সমাধান:
প্রদত্ত রাশি= $3a^3 +2a +5$
ধরি,
$f(a) = 3a^3 +2a +5 $
যদি $a= -1$ হয়, তবে আমরা পাই,
$f(-1) = 3(-1)^3 +2(-1) +5 $
বা, $f(-1) = -3 -2 +5 $
= 0

যেহেতু a = -1 হলে প্রদত্ত রাশিটির মান শূন্য হয়। সেহেতু, $(a+1)$ উহার একটি উৎপাদক।
অতএব,আমরা পাই,
$f(a)= 3a^3 +2a +5 $
= $3a^2(a+1) -3a(a+1) +5(a+1)$
= $(a+1)(3a^2-3a+5) $
নির্ণেয় উৎপাদক = $(a+1)(3a^2-3a+5)$

সংক্ষেপে,
$3a^3 +2a +5 $
= $3a^3 +3a^2-3a^2-3a +5a +5 $
= $3a^2(a+1) -3a(a+1) +5(a+1)$
= $(a+1)(3a^2-3a+5) $
