৩.৪.৩। উৎপাদকে বিশ্লেষণ কর:
$x^3 -7xy^2 -6y^3 $
সমাধান:
প্রদত্ত রাশি= $x^3 -7xy^2 -6y^3 $
এখানে x কে চলমান এবং y কে ধ্রবক হিসাবে বিবেচনা করি।
প্রদত্ত রাশিকে x এর বহুপদী হিসাবে বিবেচনা করে,
ধরি,
$f(x) = x^3 -7xy^2 -6y^3 $
যদি $x = -y$ হয়, তবে আমরা পাই,
$f(-y) = (-y)^3 -7(-y)y^2 -6y^3 $
বা, $f(-y) = -y^3 +7y^3 -6y^3 $
= 0

যেহেতু x = -y হলে প্রদত্ত রাশিটির মান শূন্য হয়। সেহেতু, $(x+y)$ উহার একটি উৎপাদক।
অতএব,আমরা পাই,
$f(x)= x^3 -7xy^2 -6y^3$
= $x^2(x+y)-xy(x+y) -6y(x+y)) $
= $(x+y)(x^2-xy-6y) $
নির্ণেয় উৎপাদক = $(x+y)(x^2-xy-6y)$

সংক্ষেপে,
$x^3 -7xy^2 -6y^3 $
= $x^3 +x^2 y-x^2 y -xy^2-6xy^2-6y^3 $
= $x^2(x+y)-xy(x+y) -6y(x+y)) $
= $(x+y)(x^2-xy-6y) $
