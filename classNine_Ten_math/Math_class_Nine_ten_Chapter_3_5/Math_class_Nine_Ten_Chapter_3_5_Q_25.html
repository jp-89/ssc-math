৩.৫.২৫। বনভোজনে যাওয়ার জন্য 5700 তাকে একটি বাস ভাড়া কর হলো এবং শর্ত হলো যে, প্রত্যেক যাত্রী সমান ভাড়া বহন করবে। 5 জন যাত্রী না যাওয়ায় মাথাপিছু ভাড়া 3 টাকা বৃদ্ধি পেল। বাসে কতজন যাত্রী গিয়েছিল ?
সমাধান:
মনে করি,
যাত্রী সংখ্যা = x
মাথাপিছু ভাড়া = ${ 5700}/x $
5 জন যাত্রী না যাওয়ায় মাথাপিছু ভাড়া = ${ 5700}/{x-5} $
মাথাপিছু ভাড়া বৃদ্ধি = $ { 5700}/{x-5} - { 5700}/x $
শর্তানুসারে,
${ { 5700}/{x-5}-{5700}/x = 3 $

বা, ${ 5700x - 5700x+ 5700×5 }/{x(x-5)} = 3 $

বা, ${ 5700×5 }/{x^2-5x} = 3 $
বা, ${ 5700×5 }= 3x^2-15x $
বা, $ 3x^2-15x - 28500 = 0 $
বা, $ x^2-95x+100x - 950 = 0 $
বা, $ x(x-95)+100(x - 95) = 0 $
বা, $ (x-100)(x-95) = 0 $
বা, $ (x-100)= 0 $
বা, $ x= -100 $ গ্রহণ যোগ্য নয়।

বা, $ (x-95)= 0 $
বা, $ x= 95 $

নির্ণেয়:
 যাত্রী সংখ্যা 95 জন।
