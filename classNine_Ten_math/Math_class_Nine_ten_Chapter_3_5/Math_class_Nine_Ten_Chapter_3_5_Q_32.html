৩২। ক,খ,ও গ এর মধ্যে 260 টাকা এরূপে ভাগ করে দাও যেন ক এর অংশের দ্বিগুন, খ এর অংশের 3 গুন এবং গ এর অংশের 4 গুন পরস্পর সমান হয়।
সমাধান:
ধরি ,
ক এর অংশ = x
ক এর অংশের দ্বিগুন, খ এর অংশের 3 গুনের সমান।
অর্থাৎ খ এর অংশ =${ 2x}/3 $
ক এর অংশের দ্বিগুন, গ এর অংশের 4 গুনের সমান
গ এর অংশ =${ 2x}/4 $
শর্তানুসারে,
$ x+ { 2x}/3 +{ 2x}/4 = 260 $

বা, $ { 12+8x+6x}/{12} = 260 $

বা, $ { 26x}/{12} = 260 $

বা, $ { 26x} = {260×12} $

বা, $ x = {260×12}/{26} $

বা, $ x = 120 $

অতএব,
ক এর অংশ = x = 120 টাকা।
খ এর অংশ =${ 2x}/3 $ = ${ 2×120}/3 $ = 80 টাকা।
গ এর অংশ =${ 2x}/4 $ = ${ 2×120}/4 $ = 60 টাকা।
উত্তর:
ক এর অংশ = 120 টাকা।
