৩.৫.৩৮। শতকরা বার্ষিক 5 টাকা হার মুনাফায় কত টাকা 12 বছরে সবৃদ্ধিমূল 1248 টাকা হবে?
সমাধান:
আমরা জানি,
সরল মুনাফার ক্ষেত্রে $ I = Pnr $

A = P(1+nr)
 এখানে,
I = n সময় পরে মুনাফা
n = নির্দিষ্ট সময়
P = মূলধন
r = মুনাফার হার
A = n সময় পরে মুনাফা সহ মূলধন বা সবৃদ্ধিমূল


এখানে, দেওয়া আছে,
সবৃদ্ধিমূল A = 1248
মুনাফার হার r = 5% = .05
নির্দিষ্ট সময় n =12
মূলধন P = ?
মান বসিয়ে পাই:
$ 1248 = P(1+12×.05) $
বা, $ 1248 = P(1+ .60) $
বা, $ P = {1248}/{1.60} $

বা, $ P = {1248}/{1.60} $

বা, $ P = 780 $

নির্ণেয়:
 মূলধন = 780.00 টাকা
