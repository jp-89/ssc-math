৩.৫.৩৯। 5% হার মুনাফায় 8000 টাকার 3 বছরের সরল মুনাফা ও চক্রবৃদ্ধি মুনাফার পার্থক্য নির্ণয় কর।
সমাধান:
আমরা জানি,
সরল মুনাফার ক্ষেত্রে $ I = Pnr $

A = P(1+nr)
চক্রবৃদ্ধি মুনাফার ক্ষেত্রে $ A = P(1+r)^n $

 এখানে,
I = n সময় পরে মুনাফা
n = নির্দিষ্ট সময়
P = মূলধন
r = মুনাফার হার
A = n সময় পরে মুনাফা সহ মূলধন

5% হার মুনাফায় কোনো টাকার 3 বছরের সরল মুনাফা:
$ I = Pnr $
= $8000×3×5/{100} $
= $1200 $ টাকা

5% হারে 3 বছরের চক্রবৃদ্ধি মুনাফার ক্ষেত্রে:
$ A = P(1+r)^n $

বা, $ A = 8000(1+5/{100})^3 $
= 8000(1+.05)^3 $
= 8000(1.05)^3 $
= 8000(1.157625) $
=9261.00 টাকা

চক্রবৃদ্ধি মুনাফা = 9261-8000
=1261.00 টাকা

উভয় মুনাফার পার্থক্য =1261-1200
= 61 টাকা

নির্ণেয়:
 উভয় মুনাফার পার্থক্য = 61 টাকা
