৩.৫.৪০। মিষ্টির উপর মূল্য সংযোজন কর (VAT ) x% . একজন বিক্রেতা ভ্যাটসহ P টাকার মিষ্টি বিক্রয় করলে তাকে কত ভ্যাট দিতে হবে?
x = 15, P = 2300 হলে, ভ্যাটের পরিমান কত?
সমাধান:
আমরা জানি,
$ S = C(1+r/100) $
এখানে,
S = বিক্রয়মূল্য = P
C = ক্রয়মূল্য
r = লাভ বা ক্ষতির হার = x
$ P = C(1+x/100) $
$ P = C({100+x}/{100}) $
$ C = {100P}/{(100+x) }$
ভ্যাটের পরিমান= P- C
= $ P - {100P}/{(100+x) } $

= $ {100P +Px- 100P}/{(100+x) } $
= $ {Px}/{(100+x) } $

x = 15, P = 2300 হলে, ভ্যাটের পরিমান কত?
ভ্যাটের পরিমান = $ {px}/{(100+x) } $

= $ {2300×15}/{(100+15) } $

= $ {2300×15}/{(115) } $

= 300 টাকা
