সমাধান:
ক. সংখ্যাটিকে x চলকে প্রকাশ করে তথ্যকে একটি সমীকরণের মাধমে প্রকাশ কর।

মনে করি সংখ্যাটি = x
অতএব, গুণাত্মক বিপরীত সংখ্যা = $1/x$
শর্তানুসারে, $x+1/x = 3$

খ. $x^3-1/x^3$ এর মান নির্ণয় কর।
$(x-1/x)^2 =(x+1/x)^2-4x. 1/x $
= $3^2 -4$
= $9-4$
= $5$
$x-1/x = √5$

$x^3-1/x^3 = (x-1/x)^3+3x.1/x( x-1/x)$
=$(√5)^3+3√5)$
= $5√5+3√5$
= $8√5$

$x^3+1/x^3 = (x+1/x)^3-3x.1/x( x+1/x)$
=$(3)^3-3(3)$
= $27-9$
= $18$


গ. প্রমান কর যে, $x^5-1/x^5 = 123$
$x^2+1/x^2 =(x+1/x)^2-2x. 1/x $

= $(3)^2-2 $

= $9-2$ = $7$

এখন
$(x^3-1/x^3)(x^2+1/x^2 ) = x^5+x-1/x-1/x^5 $

= $x^5-1/x^5+(x-1/x)$

অতএব,
$x^5-1/x^5 = (x^3-1/x^3)(x^2+1/x^2 -(x-1/x)$

= $(8√5) ×7-√5$
= $56√5- √5$
= $55√5 $
