৪.১.১৬। যদি $a^x = b, b^y = c $ এবং $ c^z=a $ হয়, তবে দেখাও যে, xyz = 1
সমাধান:
দেওয়া আছে,
$ a^x = b $
$b^y = c $ এবং
$ c^z = a $

এখন ,
$ c^z = a $

c এর মান বসিয়ে পাই,
$ {(b^y)}^z = a $
বা, $ {b}^{yz}= a $

b এর মান বসিয়ে পাই,
$ {(a^x)}^{yz}= a $
বা, $ {a}^{xyz}= a $
বা, $ {a}^{xyz}= a^1 $

সুতরাং, আমরা পাই,
xyz = 1 (প্রমাণিত )
