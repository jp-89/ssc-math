৪.১.২২। $ X= (2a^{-1} +3b^{-1})^{-1} $
$ Y = √^{pq}{ x^p/x^q } × √^{qr}{ x^q/x^r }× √^{rp}{x^r/x^p }$এবং
$ Z={ { 5^{m+1}}/{ (5^m)^{m-1}} } $ ÷ $ { { 25^{m+1}} / { (5^{m-1})^{m+1} } }$
ক) $ X $ এর মান নির্ণয় কর।
খ) দেখাও যে, $ Y + √^{4}{ 81} = 4 $

গ) দেখাও যে, $ Y ÷ Z = 25 $

৪.১.২১- ক) $ X $ এর মান নির্ণয় কর।

সমাধান:
দেওয়া আছে,
$ X = (2a^{-1} +3b^{-1})^{-1} $
 = $ (2/a +3/b)^{-1} $

= $ ({2b+3a}/{ab})^{-1} $

= $ {ab}/{2b+3a} $

অতএব, নির্ণেয় মান = $ {ab}/{2b+3a} $


৪.১.২১- খ) দেখাও যে, $ Y + √^{4}{ 81} = 4 $
সমাধান:
দেওয়া আছে,
$ Y = √^{pq}{ x^p/x^q } × √^{qr}{ x^q/x^r }× √^{rp}{x^r/x^p }$

 = $ √^{pq}{ x^{p-q} } × √^{qr}{ x^{q-r} }× √^{rp}{x^{r-p} }$

= $ x^{{p-q}/{pq} } × x^{{q-r} /{qr}}× x^{{r-p}/{rp} }$

= $ x^{{pr-qr+pq-pr+qr-pq}/{pqr} } $

= $ x^{0 } $

= $ 1 $



এখন প্রদত্ত বামপক্ষ =
$ Y + √^{4}{ 81}$

= $ Y + √^{4}{ 3^4}$

= $ Y + 3^{4/4}$

= $ Y + 3^{1}$

= $ Y + 3$
মান বসিয়ে পাই,
= $ 1 + 3$
= $ 4$

অতএব, বামপক্ষ = ডানপক্ষ (প্রমাণিত)


গ) দেখাও যে, $ Y ÷ Z = 25 $
সমাধান:
দেওয়া আছে,
$ Z={ { 5^{m+1}}/{ (5^m)^{m-1}} } $ ÷ $ { { 25^{m+1}} / { (5^{m-1})^{m+1} } } $

= ${ { 5^{m+1}}/{ 5^{m^2-m}} } $ ÷ $ { { (5^2)^{m+1}} / { 5^{(m-1)(m+1)} } } $

= ${ { 5^{m+1}}/{ 5^{m^2-m}} } $ ÷ $ { { 5^{2m+2}} / { 5^{m^2-1} } } $

= ${ { 5^{m+1-m^2+m} } $ ÷ $ { 5^{2m+2-m^2+1} } $

= ${ { 5^{2m+1-m^2} } $ ÷ $ { 5^{2m+3-m^2} } $

= $ { 5^{2m+1-m^2} } / { 5^{2m+3-m^2} } $

= $ 5^{2m+1-m^2-2m-3+m^2} $

= $ 5^{-2} $

= $ 1/{5^2} $

= $ 1/{25} $

এখন প্রদত্ত বামপক্ষ =
$ Y ÷ Z $

মান বসিয়ে পাই,
= $ {1} ÷ {1/{25}}$
= $ 1 × 25$
= $ 25$

অতএব, বামপক্ষ = ডানপক্ষ (প্রমাণিত)
