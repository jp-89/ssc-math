৪.১.৪।সরল কর : $ ( 2a^ { -1 } + 3b^ { - 1 } ) ^{ - 1 } $
সমাধান:
প্রদত্ত রাশী = $ ( 2a^ { -1 } + 3b^ { - 1 } ) ^{ - 1 } $
=$ 1/{2/a + 3/b} $

= $ 1/{{2b + 3a}/{ab}} $

= ${ab}/{3a +2b} $

=$ {ab}/{3a +2b} $ (উত্তর)

৪.১.৫।সরল কর : $ ( { a^2 b^{-1}}/{ a^{-2}b} )^2 $
সমাধান:
প্রদত্ত রাশী = $ ( { a^2 b^{-1}}/{ a^{-2}b} )^2 $
=$ ( { a^2 b^{-1}}/{ a^{-2}b^1} )^2 $

= $ ( { a^{2+2} }/{ b^{1+1}} )^2 $

= $( a^4 / b^2 )^2 $

=$ a^8/b^4 $ (উত্তর)

৪.১.৬।সরল কর : $ √{x^{-1}.y}.√{y^{-1}.z}.√{z^{-1}.x} $
সমাধান:
প্রদত্ত রাশী = $ √{x^{-1}.y}.√{y^{-1}.z}.√{z^{-1}.x} $
=$ x^{{-1}/2}.y^{1/2} .y^{{-1}/2}.z^{1/2}.z^{{-1}/2}.x^{1/2} $

= $ x^{{-1}/2 +1/2} .y^{{-1}/2 +1/2}.z^{{-1}/2 +1/2} $

= $ x^0 .y^0 .z^0$

= $ 1 .1 .1$

=$ 1 $ (উত্তর)

বিকল্প সমাধান :
প্রদত্ত রাশী = $ √{x^{-1}.y}.√{y^{-1}.z}.√{z^{-1}.x} $
উপরোক্ত সমীকরণকে আমরা নিম্নরূপে লিখতে পারি
$ √{x^{-1}.y.y^{-1}.z.z^{-1}.x }$

=$ √{x^{-1+1}.y^{-1+1}.z^{-1+1}} $

=$ √{x^0.y^0.z^0} $

=$ √{1.1.1} $

=$ 1$

৪.১.৭।সরল কর : $ {2^{n+4} - 4.2^{n+1}}/{ 2^{n+2} ÷ 2 } $
সমাধান:
প্রদত্ত রাশী = $ {2^{n+4} - 4.2^{n+1}}/{ 2^{n+2} ÷ 2 } $
=$ {2^{n+4} - {2^2}.2^{n+1}}/{ 2^{n+2} ÷ 2 } $

= $ {2^{n+4} - 2^{n+1+2}}/{{ 2^{n+2}}/2 } $

= $ {2^{n+4} - 2^{n+3}}/{{ 2^{n+2-1}} }$

= $ {2^{n+4} - 2^{n+3}}/{{ 2^{n+1}} }$

= $ 2^{n+4}/ 2^{n+1} - 2^{n+3}/2^{n+1}$

= $ 2^{n+4-n-1} - 2^{n+3-n-1}$

= $ 2^3 - 2^2$

= $ 8 -4$

= $ 4 $

=$ 4 $ (উত্তর)
