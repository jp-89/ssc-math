৪.২.২। মান নির্ণয় কর:
(ক) $log_5 \text" "\x = 3 $
 সমাধান:
প্রদত্ত রাশী = $log_5 \text" "\x = 3 $
আমরা জানি,
$y = log_a \text" "\N ...(1)$
বা, $a ^y = N $ ... (2)

প্রদত্ত রাশিকে সূত্রের সাথে তুলনা করে পাই,
a = 5
y = 3
N = x
সুতরাং, প্রদত্ত রাশিকে $a^y = N $ আকারে নিম্নরূপে লিখতে পারি,
$ 5^3 = x $
বা, $ x = 5^3 $
বা, $ x = 125 $
∴ নির্ণেয় $x = 125$

৪.২.২। মান নির্ণয় কর:
(খ) $log_x \text" "\25 = 2$
 সমাধান:
প্রদত্ত রাশী = $log_x \text" "\25 = 2 $
আমরা জানি,
$y = log_a \text" "\N ...(1)$
বা, $a ^y = N $ ... (2)

প্রদত্ত রাশিকে সূত্রের সাথে তুলনা করে পাই,
a = x
y = 2
N = 25
সুতরাং, প্রদত্ত রাশিকে $a^y = N $ আকারে নিম্নরূপে লিখতে পারি,
$ x^2 = 25 $
বা, $ x = ±5 $

এখানে x হল log এর ভিত্তি এবং log এর ভিত্তি কখনও নেগেটিভ হতে পারে না।
সুতরাং, $ x = 5 $
∴ নির্ণেয় $x = 5$

প্রদত্ত রাশিকে সূত্রের সাথে তুলনা করে পাই,

N = $1/{16}$
সুতরাং, প্রদত্ত রাশিকে $a^y = N $ আকারে নিম্নরূপে লিখতে পারি,
$ x^{-2} = 1/{16} $

বা, $ 1/x^2 = 1/4^2 $

বা, $ x^2 = 4^2 $

বা, $ x = ±4 $
