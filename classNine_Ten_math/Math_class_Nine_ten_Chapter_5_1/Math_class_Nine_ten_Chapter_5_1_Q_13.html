৫.১.১৩। সমাধান সেট নির্ণয় কর: $ {x + a}/{x - b} = {x + a}/{x + c}$
সমাধান:
দেওয়া আছে,
$ {x + a}/{x - b} = {x + a}/{x + c} $

বা, $(x + a)(x + c) = (x + a)(x - b) $

বা, $ (x + a)(x + c) - (x + a)(x - b) = 0 $

বা, $ (x + a)\{ (x + c) - (x - b)\} = 0 $

বা, $ (x + a)(x + c - x + b) = 0 $

বা, $ (x + a)(b + c) = 0 $

$ :. b + c != 0$

$ :. x + a = 0$

$ :. x = - a 0$

:. নির্ণেয় সমাধান সেট $S= \{- a \}$
