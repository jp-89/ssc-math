৫.১.১৯। সমাধান সেট নির্ণয় কর: $ {x+2b^2+c^2}/{a+b} + {x+2c^2+a^2}/{b+c}+ {x+2a^2+b^2}/{c+a} = 0$
সমাধান:
দেওয়া আছে,
$ {x+2b^2+c^2}/{a+b} + {x+2c^2+a^2}/{b+c}+ {x+2a^2+b^2}/{c+a} = 0 $

বা, ${x+2b^2+c^2}/{a+b} +(a-b)+ {x+2c^2+a^2}/{b+c}+(b-c)+ {x+2a^2+b^2}/{c+a}+(c-a) = 0 $

বা, ${x+2b^2+c^2+a^2-b^2}/{(a+b)}+ {x+2c^2+a^2+b^2-c^2}/{(b+c)}+ {x+2a^2+b^2+c^2-a^2}/{(c+a)} = 0 $

বা, ${x+a^2+b^2+c^2}/{(a+b)}+ {x+a^2+b^2+c^2}/{(b+c)}+ {x+a^2+b^2+c^2}/{(c+a)} = 0 $

বা, ${(x+a^2+b^2+c^2)} \{{1}/{(a+b)}+ {1}/{(b+c)}+ {1}/{(c+a)}\} = 0 $

এখানে, $\{{1}/{(a+b)}+ {1}/{(b+c)}+ {1}/{(c+a)}\} ≠ 0 $

অতএব,
$(x+a^2+b^2+c^2) = 0$
বা, $x = -(a^2+b^2+c^2) $
:. নির্ণেয় সমাধান সেট $S= -(a^2+b^2+c^2)$
