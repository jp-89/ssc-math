৫.১.২০। একটি সংখ্যা অপর একটি সংখ্যার $2/5$ গুনা। সংখ্যা দুইটির সমষ্টি 98 হলে, সংখ্যা দুইটি নির্ণয় কর।
সমাধান:
ধরি, একটি সংখ্যা = x
অতএব,অপর সংখ্যাটি = ${2x}/5$
শর্তানুসারে,
বা,$ {5x+2x}/5 = 98 $
বা,$ {7x}/5 = 98 $
বা, $ {7x}/5 = 98 $
বা, $ 7x = 98 × 5$
বা, $ x = { 98 × 5}/7$
বা, $ x = { 14 × 5}/7$
বা, $ x =70$
অতএব,একটি সংখ্যা = 70 এবং
অপর সংখ্যাটি = ${2x}/5$ = ${2×70}/5$
= ${2×14}$
= 28
