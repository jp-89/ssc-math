৫.১.২১। একটি প্রকৃত ভগ্নাংশের লব ও হরের অন্তর 1; লব থেকে 2 বিয়োগ করলে ও হরের সাথে 2 যোগ করলে যে ভগ্নাংশ পাওয়া যায় তা $1/6$ এর সমান। ভগ্নাংশটি নির্ণয় কর।
সমাধান:
ধরি, হর = x
সুতরাং, লব = x-1
লব থেকে 2 বিয়োগ করে প্রাপ্ত লব = x-1 - 2 = x -3
হরের 2 সাথে যোগ করে প্রাপ্ত হর = x+2
এখন প্রাপ্ত ভগ্নাংশ = ${ লব }/{হর } = {x-3}/{x+2} $
শর্তানুসারে,
বা,$ {x-3}/{x+2} = 1/6 $
বা,$ 6x-18 = x+2$
বা, $ 6x-x =2+18 $
বা, $ 5x = 20$
বা, $ x = { 20}/5$
বা, $ x = 4 $
বা, $ x =4 $
অতএব, হর = 4 এবং
লব = x-1 = 4-1 = 3
ভগ্নাংশ = ${ লব }/{হর } = {3}/{4} $
অতএব নির্ণেয় ভগ্নাংশটি $ {3}/{4} $
