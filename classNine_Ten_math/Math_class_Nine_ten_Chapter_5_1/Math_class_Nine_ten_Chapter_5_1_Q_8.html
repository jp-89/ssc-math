৫.১.৮। সমাধান কর: $ {x - a}/b + {x - b}/a + {x - 3a - 3b}/{a + b} = 0 $
সমাধান:
দেওয়া আছে,
$ {x - a}/b + {x - b}/a + {x - 3a - 3b}/{a + b} = 0 $

বা, $ (x - a)/b - 1) + ((x - b)/a - 1) + ((x - 3a - 3b)/(a + b) + 2) = 0 $

বা, $ (x - a - b)/b + (x - b - a)/a + (x - 3a - 3b + 2a + 2b)/(a + b) = 0 $

বা, $ (x - a - b)/b + (x - a - b)/a + (x - a - b)/(a + b) = 0 $

বা, $ (x - a - b)( 1/b + 1/a + 1/(a + b)) = 0 $

বা, $ x - a - b = 0 $

বা, $ :. x = a + b $

:. নির্ণেয় সমাধান x = a + b
