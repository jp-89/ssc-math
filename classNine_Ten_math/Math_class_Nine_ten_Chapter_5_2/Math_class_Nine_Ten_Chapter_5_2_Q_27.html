দৈর্ঘ্য= x এবং
প্রস্থ = y
অতএব ক্ষেত্রফল = xy
শর্তানুসারে,
$ xy = 192 $
বা, $ y = {192}/x $ ...(1)
আবার শর্তানুসারে,
$ (x-4)(y+4) = 192 $
বা, $ xy+4x-4y-16 = 192 $
মান বসিয়ে পাই,
বা, $ 192+4x-4({192}/x)-16 = 192 $
বা, $ 4x-4({192}/x)-16 = 0 $
বা, $ x-({192}/x)-4 = 0 $
বা, $ x^2-192-4x = 0 $
বা, $ x^2-4x-192 = 0 $
বা, $ x^2-16x+12x-192 = 0 $
বা, $ x(x-16)+12(x-16) = 0 $
বা, $ (x+12)(x-16) = 0 $
বা, $ x+12=0 $
বা, $ x =-12 $ (গ্রহণযোগ্য নয়)

বা, $ x-16 =0 $
বা, $ x = 16 $

বা, $ y = {192}/16 = 12 $
