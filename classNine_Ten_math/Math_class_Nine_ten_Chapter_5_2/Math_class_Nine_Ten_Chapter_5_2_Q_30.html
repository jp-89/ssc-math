সমাধান:
মনে করি,
ছাত্র ছাত্রীর সংখ্যা = x
প্রত্যেকের সহপাঠীর সংখ্যা = x-1
প্রত্যেকের চাঁদার পরিমান = x-1 টাকা
অতএব, মোট চাঁদার পরিমান = x(x-1) টাকা

শর্তানুসারে,
$ x(x-1) = 420 $
বা, $ x^2-x = 420$
বা, $ x^2-x -420 = 0 $
বা, $ x^2-21x+20x -420 = 0 $
বা, $ x(x-21)+20(x -21) = 0 $
বা, $ ( x+20)(x-21) = 0 $
বা, $ x+20 = 0 $
বা, $x = -20 $ (গ্রহণযোগ্য নয়)

বা, $ x-21 = 0 $
বা, $ x = 21 $

অতএব,
ছাত্র ছাত্রীর সংখ্যা =21 জন
প্রত্যেকের চাঁদার পরিমান = 21-1 = 20 টাকা
