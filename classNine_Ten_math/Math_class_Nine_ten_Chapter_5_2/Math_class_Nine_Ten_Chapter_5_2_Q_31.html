= $ {x(x+30)}/100 $ টাকা

শর্তানুসারে,
$ {x(x+30)}/100 = 70 $
বা, $ x^2+30x = 7000$
বা, $ x^2+30x-7000 = 0 $
বা, $ x^2+100x-70x-7000 = 0 $
বা, $ x(x+100)-70(x+100) = 0 $
বা, $ ( x+100)(x-70) = 0 $
বা, $ x+100 = 0 $
বা, $x = -100 $ (গ্রহণযোগ্য নয়)

বা, $ x-70 = 0 $
বা, $ x = 70 $
