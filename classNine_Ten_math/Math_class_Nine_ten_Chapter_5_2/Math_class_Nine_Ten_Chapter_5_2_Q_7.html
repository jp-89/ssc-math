৫.২.৭। $ x^2-( a+b)x+ab=0 $ সমীকরণের সমাধান সেট নিচের কোনটি?
ক. { a, b}
খ. { a, -b }
গ. { -a, b}
ঘ.{-a, -b }

সমাধান:

$ x^2-( a+b)x+ab=0 $
$ x^2-( a+b)x+ab=0 $
বা, $ x^2 -ax-bx+ab = 0$
বা, $ x(x-a)-b(x-a)=0$
বা, $ (x-a)(x-b)=0$
বা, $ (x-a)=0$
বা, $ x= a $
এবং
$ x -b= 0 $
বা, $ x = b $

সমাধান সেট = {a,b}
নির্ণেয়:
ক. { a, b}
