১৮। সমাধান কর : $ {cosA - SinA}/{cosA + sinA} = {√3 -1}/ {√3 +1} $
সমাধান:
দেওয়া আছে,
$ {cosA - SinA}/{cosA + sinA} = {√3 -1}/ {√3 +1} $

আমরা জানি ,
$ a/b = c/d $ হলে, আমরা নিম্নরূপে প্রকাশ করতে পারি
$ {a +b }/{a -b }= {c + d}/{ c- d} $

 উপরোক্ত নিয়ম প্রয়োগ করে লিখতে পারি,
${(cosA - SinA) + (cosA + sinA)}/{(cosA - sinA) - (cosA + sinA) } = {(√3 -1 )+ (√3 +1)}/ {√3 -1 -(√3 +1)} $

বা , ${2cosA }/{- 2sinA } = {2√3 }/ { -2 } $

বা , ${cosA }/{sinA } = {√3 }/ { 1} $

বা , $cotA = √3 $

বা , $A = 30 $ [উভয় পক্ষে $ cot^{-1} $ নিয়ে ]
