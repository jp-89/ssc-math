২১। সমাধান কর : $ sinθ + cosθ = 1 \text" যখন " 0 ^◦ ≤θ ≤ 90 $

সমাধান:
দেওয়া আছে,
$ sinθ + cosθ = 1 \text"      " 0 ^◦ ≤θ ≤ 90 $
উভয় পক্ষকে বর্গ করিয়া পাই,
$ (sinθ + cosθ)^2 = (1)^2 $
বা, $ sin^2θ + cos^2θ + 2SinθCosθ = 1 $
বা, $ 1 + 2SinθCosθ = 1 $
বা, $ 2SinθCosθ = 1-1 $
বা, $ 2SinθCosθ = 0 $
বা, $ SinθCosθ = 0 $
সুতরাং, আমরা পাই,
Sinθ = 0
Sinθ = Sin0
θ = 0
আবার, আমরা পাই,
Cosθ = 0
বা, Cosθ = 0
$Cosθ = cos90^০$
$θ = 90^০$
অতএব,প্রাপ্ত সমাধান,
$θ = 0^০, 90^০$
