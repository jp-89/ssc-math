সৃজনশীল -১৩.৬। $ 33 + 29 +25 +....-19 $ একটি ধারা এবং
$ m ={ √{1+y}+ √{1-y} }/ { √{1+y}- √{1-y} } $
ক) ধারাটির 12 তম পদ কত ?
খ) প্রমান কর যে, $ m^2-{2m}/y +1 =0 $
গ) ধারাটির ১ম পদকে ১ম পদ এবং সাধারণ অন্তরকে সাধারণ অনুপাত ধরে একটি গুণোত্তর ধারা গঠন কর এবং ধারাটির ১ম পাঁচটি পদের সমষ্টি নির্ণয় কর।

ক) ধারাটি নির্ণয় কর।
সমাধান:
ধারাটির প্রথম পদ $a = 33 $
সাধারণ অন্তর $d = 29-33$
$-4$

আমরা জানি,
সমান্তর ধারার n তম পদ = $ {a + (n-1)d} $
অতএব,
12 তম পদ = $ {33 + (12-1)(-4) $
= $ {33 + (12-1)(-4) } $
= $ {33 + (11)(-4) } $
= $ {33 -44 } $
= $ {-11 } $
অতএব,
ধারাটির 12 তম পদ = $ -11 $

খ) প্রমান কর যে, $ m^2-{2m}/y +1 =0 $
দেওয়া আছে,
$ m ={ √{1+y}+ √{1-y} }/ { √{1+y}- √{1-y} } $

যোজন বিয়োজন করিয়া পাই,
বা, ${ m+1}/{m-1} ={ √{1+y}+ √{1-y}+ √{1+y}- √{1-y} }/ { √{1+y}+ √{1-y}-( √{1+y}- √{1-y} )} $

বা, ${ m+1}/{m-1} ={ 2√{1+y} }/ { 2√{1-y}} $

বা, ${ m+1}/{m-1} ={ √{1+y} }/ { √{1-y}} $

উভয় পক্ষকে বর্গ করিয়া পাই,
বা, $ ({ m+1}/{m-1})^2 =({ √{1+y} }/ { √{1-y}} )^2 $

বা, $ { m^2+2m+1}/{m^2-2m+1} ={1+y} / { 1-y} $

যোজন বিয়োজন করিয়া পাই,
বা, $ { m^2+2m+1 + m^2-2m+1}/{ m^2+2m+1 -(m^2-2m+1)} ={1+y+1-y} / { 1+y -(1-y)} $

বা, $ { 2m^2+2 }/{ 4m} ={2} / { 2y} $

বা, $ { m^2+1 }/{ 2m} ={1} / { y} $

বা, $ { m^2+1 } ={2m} / { y} $
বা, $ m^2-{2m}/y +1 =0 $ (প্রমাণিত)

গ) ধারাটির ১ম পদকে ১ম পদ এবং সাধারণ অন্তরকে সাধারণ অনুপাত ধরে একটি গুণোত্তর ধারা গঠন কর এবং ধারাটির ১ম পাঁচটি পদের সমষ্টি নির্ণয় কর।
ধারাটির প্রথম পদ $a = 33 $ এবং
সাধারণ অনুপাত $r = -4 $

এখানে r <1
সুতরাং, এটি একটি গুণোত্তর ধারা।

আমরা জানি,
গুণোত্তর ধারার n তম পদ = $ ar^{n-1}$

অতএব,
দ্বিতীয় পদ = $ 33(-4)^{2-1} $
=$ 33×(-4)^1 $
=$ 33×(-4) $
=$ -132 $

তৃতীয় পদ = $ 33(-4)^{3-1} $
=$ 33×(-4)^2 $
=$ 33×(16) $
=$ 528 $

চতুর্থ পদ = $ 33(-4)^{4-1} $
=$ 33×(-4)^3 $
=$ 33×(-64) $
=$ -2112 $

অতএব,
ধারাটি = $ 33-132+528-2112+..... $

আমরা জানি, গুণোত্তর ধারার প্রথম n পদের সমষ্টি
$ S_n = {a(1-r^n )}/{1-r}$

মান বসিয়ে পাই
$ S_{5} = {33\{1-(-4)^{5} \}}/{\{1-(-4)\}}$

= ${33\{1+(4)^{5}\} }/{1+4}$

= ${33(1+1024 )}/{5}$

= ${33(1025 )}/{5}$

= ${33(205 )}$

= ${6765}$

অতএব,
প্রথম 5 টি পদের সমষ্টি = $6765$
