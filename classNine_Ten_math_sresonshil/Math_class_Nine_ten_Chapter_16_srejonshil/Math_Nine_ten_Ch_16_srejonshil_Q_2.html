সৃজনশীল -১৬.২। নাটোর উত্তরা গণভবনের ভিতর একটি আয়তাকার বাগানের দৈর্ঘ্য 60 মি. এবং প্রস্থ 40 মি.। বাগানের ভিতর সমান পাড় বিশিষ্ট একটি পুকুর আছে। পুকুরের ক্ষেত্রফল বাগানের ক্ষেত্রফলের এক তৃতীয়াংশ।
ক) বাগানের কর্ণের দৈর্ঘ্য নির্ণয় কর।
খ) পুকুরের পরিসীমা নির্ণয় কর।
গ) পুকুরের পরিসীমার সমান পরিসীমা বিশিষ্ট একটি বর্গক্ষেত্রের মেঝে 30 সে.মি. বর্গাকার টালী দিয়ে বাঁধতে মোট কতটি টালী লাগবে?

ক) বাগানের কর্ণের দৈর্ঘ্য নির্ণয় কর।
সমাধান:
বাগানের কর্ণের দৈর্ঘ্য = $√{ \text" দৈর্ঘ্য " ^2 + \text" প্রস্থ "^2 } $
= $√{ 60^2 + 40^2 } $
= $√{ 3600 + 1600 } $
= $√{ 5200 } $
= $72.11 $ মিটার


অতএব,
বাগানের কর্ণের দৈর্ঘ্য = $ 72.11 $ মিটার

খ) পুকুরের পরিসীমা নির্ণয় কর।
বাগানের ক্ষেত্রফল = $60×40 $ বর্গমিটার
= $2400 $ বর্গমিটার।

অতএব,
পুকুরের ক্ষেত্রফল = $ 1/3×2400 $ বর্গমিটার
= $800 $ বর্গমিটার

মনে করি পুকুরের পাড় x মিটার চওড়া।
অতএব,
পুকুরের দৈর্ঘ্য = $60 - 2x $ মিটার।
পুকুরের প্রস্থ = $40 - 2x $ মিটার।

অতএব,
পুকুরের ক্ষেত্রফল = $(60 - 2x ) ( 40 - 2x ) $ বর্গমিটার
= $(60 - 2x ) ( 40 - 2x ) $ বর্গমিটার
= $2400 - 120x -80x +4x^2 $ বর্গমিটার
= $2400 - 200x +4x^2 $ বর্গমিটার
= $4x^2 -200x+2400 $ বর্গমিটার

শর্তানুসারে,
$4x^2 -200x+2400 = 800 $
বা, $4x^2 -200x+2400 - 800 =0$
বা, $4x^2 -200x+1600 =0$
বা, $x^2 -50x+400 =0$
বা, $x^2 -40x -10x+400 =0$
বা, $x(x -40) -10(x-40) =0$
বা, $(x -40)(x -10) =0$
বা, $ x -40 =0$
বা, $ x =40$
গ্রহণযোগ্য নয় । কারণ পুকুরের পাড় বাগানের প্রস্থের সমান সমান হতে পারে না।
আবার,
বা, $ x -10 =0$
বা, $ x =10$

অতএব,

= $60 - 2×10 $ মিটার।
= $60 - 20 $ মিটার।
= $40 $ মিটার।

পুকুরের প্রস্থ = $40 - 2x $ মিটার।
= $40 - 2×10 $ মিটার।
= $40 - 20 $ মিটার।
= $20 $ মিটার।

পুকুরের পরিসীমা = 2( দৈর্ঘ্য + প্রস্থ) মিটার
= $ 2( 40 + 20 ) $ মিটার।
= $ 2×60$ মিটার।
= $ 120 $ মিটার।


গ) পুকুরের পরিসীমার সমান পরিসীমা বিশিষ্ট একটি বর্গক্ষেত্রের মেঝে 30 সে.মি. বর্গাকার টালী দিয়ে বাঁধতে মোট কতটি টালী লাগবে?
বর্গক্ষেত্রের পরিসীমা = 120 মিটার।
সুতরাং,
বর্গক্ষেত্রের প্রতিটি বাহুর দৈর্ঘ্য =${120}/4$ মিটার।
=$30$ মিটার।

= 900 বর্গমিটার।
