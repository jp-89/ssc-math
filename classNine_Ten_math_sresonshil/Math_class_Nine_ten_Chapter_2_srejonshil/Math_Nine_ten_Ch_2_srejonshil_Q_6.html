(খ) দেখাও যে, $ (A ∪ B)^' = A^' ∩ B^' $

সমাধান:
দেওয়া আছে,
$A = \{ x ∈ N :x মৌলিক সংখ্যা \text" এবং " x ≤7 \} $
এখানে, স্বাভাবিক সংখ্যার সেট $N = \{ 1,2,3 ,4,5,.......\} $

এখানে স্পষ্টতযে, 1 ও এর 7 মধ্যে এবং 7থেকে ছোট অথবা ৭ এর সমান মৌলিক সংখ্যা সমূহ হল:
$2,3,5,7$
অতএব, $ A = \{ 2,3,5,7\}$

$A ∪ B = \{ 2,3,5,7\} ∪ \{2,4,6\} $
= উভয় সেটের সম্মিলিত উপাদান সমূহ।
= $\{ 2,3,4,5,6,7\} $

$(A ∪ B)^' = U- (A ∪ B) $ = $ \{1,2,3,4,5,6,7 \} - \{ 2,3,4,5,6,7\} $
= উভয় সেটের সাধারণ উপাদান সমুহ
= $\{ 1\} $
$ A^' = U - A $
= $ \{1,2,3,4,5,6,7 \} - \{ 2,3,5,7\} $
= উভয় সেটের অন্তর
= $ \{1,4 \} $

$ B^' = U - B $
= $ \{1,2,3,4,5,6,7 \} - \{ 2,4,6\} $
= উভয় সেটের অন্তর
= $ \{1,3,5,7 \} $

$ A^' ∩ B^' = \{1,4 \} ∩ \{1,3,5,7 \} $
= উভয় সেটের সাধারণ উপাদান সমুহ
= $ \{1 \} $

অতএব, $ (A ∪ B)^' = A^' ∩ B^' $ ( প্রমাণিত)
