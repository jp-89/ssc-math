খ. $ (A /B )^{p+r} ×(B /C )^{q+p} × (C/A)^{r+q} $ এর মান নির্ণয় কর।

সমাধান:
এখন,
$ A /B = {6^{p +q}}/{6^{q +r }}$

=$6^{p +q-q-r}$

=$6^{p -r}$

$ B /C = {6^{q +r }}/{6^{r +p }}$

=$6^{q +r-r-p}$

=$6^{q -p}$

$ C /A = {6^{r +p }}/{6^{p +q}}$

=$6^{r +p-p-q}$

=$6^{r -q}$

এখন,
$ (A /B )^{p+r} ×(B /C )^{q+p} × (C/A)^{r+q} $

=$ (6^{p -r} )^{p+r} ×(6^{q -p} )^{q+p} × (6^{r -q})^{r+q} $

=$ 6^{p^2 -r^2} ×6^{q^2 -p^2} × 6^{r^2 -q^2} $

=$ 6^{p^2 -r^2+q^2 -p^2+r^2 -q^2} $

=$ 6^0 $

=$1$

গ. দেখাও যে, $(AB)^{p -r } ×(BC) ^{q -p } ×(CA ) ^{r -q } =1 $

সমাধান:
এখন,
$(AB)^{p -r }$

=$(A)^{p -r } . (B)^{p -r }$

= $(6^{p +q})^{p -r } .(6^{q +r })^{p -r }$

= $6^{p^2 -pr+pq-qr} .6^{pq -qr+pr-r^2 }$

= $6^{p^2 -pr+pq-qr + pq -qr+pr-r^2} $

=$6^{p^2-r^2+2pq-2qr}$

$(BC)^{q -p }$

=$(B)^{q -p } . (C)^{q -p }$

=$(6^{q +r })^{q -p } . (6^{r +p })^{q -p }$

=$6^{q^2-pq +qr -pr} . 6^{qr -pr+pq -p^2}$

=$6^{q^2-pq +qr -pr +qr -pr+pq -p^2} $

=$6^{q^2-p^2+2qr-2pr}$

$(CA)^{r -q }$

=$(C)^{r -q } . (A)^{r -q }$

=$(6^{r +p })^{r -q } . (6^{p +q})^{r -q }$

=$6^{r^2-qr +pr-pq } . 6^{pr -pq+qr-q^2}$

=$6^{r^2-qr +pr-pq+pr -pq+qr-q^2 } $

=$6^{r^2-q^2+2pr-2pq}$

এখন,
$6^{p^2-r^2+2pq-2qr} ×6^{q^2-p^2+2qr-2pr} ×6^{r^2-q^2+2pr-2pq} $

= $6^{p^2-r^2+2pq-2qr + q^2-p^2+2qr-2pr + r^2-q^2+2pr-2pq} $

= $6^{0} $

=$1$
