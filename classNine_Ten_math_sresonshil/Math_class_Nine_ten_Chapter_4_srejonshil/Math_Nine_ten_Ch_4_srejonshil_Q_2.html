২। L , M , N তিনটি বীজগাণিতিক রাশি যেখানে,
$ L = {x^a}/{x^b}, M = {x^b}/{x^c}, N = {x^c}/{x^a} $
ক. L = 1 হলে দেখাও যে, a = b
খ. প্রমান কর যে, $ √^{ab}{L} × √^{bc} { M} × √^{ca} { N} =1 $
গ. উদ্দীপকের আলোকে দেখাও যে, $ log_k L ^{a +b } + log_k M ^{b +c } + log_k N ^{c +a } = 0 $


ক. L = 1 হলে দেখাও যে, a = b

সমাধান:
দেওয়া আছে,
$ L = {x^a}/{x^b} $

বা, ${x^a}/{x^b}= L$

বা, $x^{a-b}= 1 $

বা, $x^{a-b}= x^0 $

অতএব,
$a-b =1 $

বা, $a=b $ (প্রমাণিত)

খ. প্রমান কর যে, $ √^{ab}{L} × √^{bc} { M} × √^{ca} { N} =1 $

সমাধান:
দেওয়া আছে বামপক্ষ=
$ √^{ab}{L} × √^{bc} { M} × √^{ca} { L}$

=$ √^{ab}{ {x^a}/{x^b}} × √^{bc} { {x^b}/{x^c}} × √^{ca} { {x^c}/{x^a}}$

=$ √^{ab}{ x^{a-b}} × √^{bc} { x^{b-c}} × √^{ca} { x^{c-a}}$

=$ x^{{a-b}/{ab}} × x^{{b-c}/{bc}} × x^{{c-a}/{ca}}$

=$ x^{{ca-bc+ab-ca+bc-ab}/{abc}} $

=$ x^{{0}/{abc}} $

=$ x^0 $

= $1 $ (প্রমাণিত)

গ. উদ্দীপকের আলোকে দেখাও যে, $ log_k L ^{a +b } + log_k M ^{b +c } + log_k N ^{c +a } = 0 $
সমাধান:
$ L = {x^a}/{x^b} $

বা, $ L = x^{a-b} $

বা, $ L^{a +b } = (x^{a-b})^{a +b } $

= $x^{a^2-b^2}$

অনুরূপভাবে,
$ M^{b +c } = x^{b^2-c^2} $

$ N ^{c +a } = x^{c^2-a^2} $

মান বসিয়ে পাই,
$ log_k L ^{a +b } + log_k M ^{b +c } + log_k N ^{c +a } $

= $ log_k x^{a^2-b^2} + log_k x^{b^2-c^2} + log_k x^{c^2-a^2} $

= $ (a^2-b^2)log_k x + (b^2-c^2)log_k x + (c^2-a^2)log_k $

= $ (a^2-b^2 + b^2-c^2 + c^2-a^2)log_k x $

$ 0 ×log_k x $

= $0$
